1.0  安裝與更新
---------------------

1.1  使用 Windows XP Service Pack 1 (SP1) 的客戶必須下載更新程式 (KB822603)

設定時，您可能會看見以下訊息：
若要安裝 Microsoft LifeCam 軟體，您必須先安裝更新。 
您可以從 Microsoft 網站下載更新 (KB822603)。

(注意：如果您未收到此訊息，這是因為安裝程式偵測到您並不需要或已經安裝此更新程式)。

若需修正此問題：
1. 請造訪 http://support.microsoft.com/support。
2. 搜尋 KB822603。
3. 按一下 [立即下載 822603 套件] 連結。
4. 請依照指示下載並安裝更新程式。

安裝更新程式之後，請重新啟動 LifeCam 安裝程式，以便安裝 LifeCam 軟體。 

---------------------

1.2  安裝錯誤訊息：硬碟空間不足或發生錯誤。

收到此錯誤訊息的原因有很多種：
- 防毒程式、快顯封鎖程式或間諜軟體偵測程式可能會導致軟體安裝被封鎖。 (若需更多資訊，請參閱 1.3。)
- 安裝軟體的硬碟空間可能已滿。  

---------------------

1.3  防毒程式、快顯封鎖程式或間諜軟體偵測程式可能會導致軟體安裝被封鎖

在安裝 LifeCam 軟體期間，您的防毒程式、快顯封鎖程式或間諜軟體偵測程式可能
封鎖軟體安裝程序。 

但是大部分的程式都會詢問您是否允許或不允許繼續安裝。 
當系統出現提示時，請選擇允許程式繼續安裝。 

若需修正此問題：
- 請嘗試暫時結束您的防毒程式，然後重新執行 LifeCam 安裝程式。

---------------------
 
1.4  若要完成自動更新安裝，請關閉 LifeCam 軟體後再重新開啟
 
下載並安裝自動更新後，您必須關閉 LifeCam 軟體然後再度開啟，
變更才會生效。

有些更新程式可能需要您重新開機，變更才會生效。

---------------------

1.5  安裝 Logitech® 網路攝影機軟體之後，請務必重新安裝 Microsoft LifeCam 軟體 (錯誤訊息代碼 39)

Logitech 網路攝影機軟體安裝過程可能導致 Microsoft LifeCam 軟體發生問題，
包括顯示錯誤訊息代碼 39。

如欲在同一台電腦上安裝 Microsoft LifeCam 軟體和 Logitech QuickCam 軟體，
請先安裝 Logitech QuickCam 軟體，再安裝 Microsoft LifeCam 軟體。  

使用 Microsoft LifeCam 時，您可能需要關閉或從系統匣中停用Logitech QuickCam 軟體。

2.0  硬體
---------------------


2.1  關於通用序列匯流排 (USB) 連接埠

若網路攝影機在目前使用的 USB 連接埠上無法正常運作，請嘗試將其插入另一個 USB 連接埠。

您的 Microsoft LifeCam 可與 USB 1.1 或 USB 2.0 連接埠搭配使用， 

- USB 1.1 和 USB 2.0 連接埠外觀相同，可能的話，請使用 USB 2.0 連接埠來連接 LifeCam 網路攝影機。
- USB 1.1 連接埠可能無法支援某些網路攝影機功能。

不同的 USB 連接埠可能會為網路攝影機提供不同程度的電力。 例如，鍵盤上的 USB 集線器可能無法為 LifeCam 網路攝影機提供足夠電力。 您可能需要將網路攝影機連接到電腦主機上的 USB 連接埠，或已連接電源的 USB 集線器上。


---------------------

2.2  若連接至電腦的 USB 裝置過多，可能會導致網路攝影機執行不當或無法執行

若有太多 USB 裝置連接至電腦，可能導致 LifeCam 網路攝影機執行不當或停止運作。 

問題可能包括：視訊區變黑 (沒有畫面或聲音)、啟動視訊通話後 Windows Live Messenger 會停止運作 (當機)，
或網路攝影機在高解析度視訊中無法運作。

若需修正此問題：
- 嘗試拔除後重新連接 LifeCam 網路攝影機。
- 嘗試拔除其他閒置的 USB 裝置。
- 嘗試連接電腦上的另一個 USB 連接埠。
- 若使用 USB 集線器，嘗試拔除集線器，然後直接將網路攝影機連接到電腦上。

---------------------

2.3  LifeCam VX-6000 可能無法搭配 ATI Radeon 9600 視訊卡使用

使用 ATI Radeon 9600 視訊卡時，LifeCam VX-6000 擷取到的影像可能定格或閃爍。 

---------------------

2.4  放大影像至全螢幕

某些視訊卡可以全螢幕方式來顯示視訊。

若不需要以全螢幕觀賞 LifeCam 視訊，通常可停用此類功能。
請參閱硬體製造商之說明文件，取得停用或關閉此功能的指示。

---------------------

2.5  在次要監視器上開啟或停止視訊時，LifeCam 可能當機

若您同時使用多個監視器，並從次要監視器來執行 LifeCam 時，LifeCam 軟體可能會當機。

若要避免此錯誤：
 - 在主監視器上執行 LifeCam 軟體。

3.0  視訊擷取
---------------------

3.1  LifeCam VX-6000 -- 當使用 USB 1.1 連接埠時，並不支援所有的解析度

連接到 USB 1.1 連接埠時，LifeCam VX-6000 可支援的最高解析度為 800 x 600 像素，
最高相片解析度為 1.9 MP。 

若需修正此問題：
- 可能的話，請使用 USB 2.0 連接埠。
  - 或 -
- 降低解析度。

---------------------

3.2  LifeCam VX-6000 -- 1.3 MP 視訊解析度不支援畫面上下左右移動、縮放與臉部追蹤等功能

目前，1.3 MP (最高) 視訊解析度不支援畫面上下左右移動、縮放與臉部追蹤等功能。
若要使用畫面上下左右移動、縮放與臉部追蹤等功能，請選擇較低的視訊解析度。

---------------------

3.3  LifeCam NX-6000 -- 預覽擷取畫面可能呈現黑色

在部分情況下，預覽擷取畫面可能無法顯示。

若需修正此問題：
- 請拔除網路攝影機，再重新連接。
  - 或 -
- 降低視訊解析度。

---------------------

3.4  使用一般視訊驅動程式時，擷取畫面保持在前景顯示狀態

在部分情況下，即使在 LifeCam 軟體視窗上方開啟其他視窗，視訊擷取畫面仍保持在前景。
這可能是因為您的視訊顯示是使用一般驅動程式。

若需修正此問題：
- 請參閱硬體製造商之說明文件，取得更新視訊驅動程式的資訊。


---------------------

3.5  部分視訊特效可能會導致電腦效能變慢

在高解析度視訊中增加某些視訊特效可能讓電腦效能變慢。

如要提高電腦效能：
- 請設定較低的視訊解析度。
  - 或 -
- 關閉視訊特效。


---------------------
4.0  即時訊息程式
---------------------

4.1  網路攝影機上的 [Windows Live Call] 按鈕無法運作

在某些情況下，在安裝或更新 LifeCam 軟體的過程中，網路攝影機上的 [Windows Live Call] 按鈕可能沒有正確初始化。 

若要修正此問題：

- 請登出電腦，再重新登入。
  - 或 - 
- 重新啟動電腦。 

---------------------

4.2  LifeCam NX-6000 -- 某些訊息軟體用戶端可能需要安裝 Microsoft LifeCam 軟體

為了確保產品與某些訊息軟體用戶端之間的相容性 (例如 Skype 和 Yahoo! Messenger)，可能需要安裝 Microsoft LifeCam 軟體。

---------------------

4.3  某些即時通訊軟體程式不能顯示所有的視訊特效

某些即時通訊軟體 (例如 Skype)，無法顯示所有的視訊特效。

---------------------

4.4  使用 America Online (AOL) 即時通訊軟體進行視訊通話可能會斷線

若您使用防火牆，使用 AOL 即時通訊軟體建立的視訊通話可能會馬上斷線。

若需更多調整防火牆或路由器的相關資訊，請參閱 www.aim.com。

5.0  其他應用程式
-----------------------

5.1  LifeCam VX-6000 不支援使用部分原生 64 位元程式 

LifeCam VX-6000 無法與部分原生 64 位元程式搭配運作。 但在大多數的情況下，它能夠
在 64 位元作業系統中與這些程式的 34 位元版本搭配運作。

---------------------

5.2  Microsoft Office Communicator 音訊和視訊調整精靈中可能找不到 LifeCam VX-6000  

在 Microsoft Office Communicator 中，使用者可透過 [音訊和視訊調整精靈] 設定 LifeCam，在部分情況下，LifeCam VX-6000 可能不會顯示在選單中。

若需修正此問題：
-  插上 LifeCam 網路攝影機之後再開啟電腦。

---------------------

5.3  張貼到 Windows Live Spaces 時，出現的是 [已經登入] 頁面或 HTML 程式碼，而非我的分享空間

若在同一台電腦上同時使用兩個不同 Windows Live Spaces 帳戶張貼照片，
Windows Live Spaces 就會出現「已經登入」錯誤。

若需修正此問題：
1.  請關閉所有 Internet Explorer 瀏覽器視窗。 
2.  在 LifeCam 軟體上，再按一下 [張貼到我的 Windows Live Spaces 部落格] 按鈕。 

>>若要避免此錯誤：

- 請關閉使用者工作階段之間的 Internet Explorer。

---------------------

5.4  若一使用者變更了預設網路攝影機，則所有使用者的預設攝影機也會隨之變更。

LifeCam 軟體設定可能會影響所有的使用者設定檔。  比如說，如果您變更了一個使用者設定檔中的預設網路攝影機，則該部電腦上所有使用者設定檔的預設網路攝影機都會因此改變。

----------------------

5.5  Microsoft LifeCam 軟體中 LifeCam NX-6000 的某些特定功能，在 Windows XP 作業系統上可能會在網路攝影機 [內容] 視窗中被截斷

Microsoft LifeCam 軟體中 LifeCam NX-6000 特定的某些功能，在 Windows XP 作業系統上可能會在網路攝影機 [內容] 視窗中被截斷，這個問題與 Windows UVC 驅動程式中的內容設置有關。當您更改設定後視訊品質不如預期效果，仍可按 [預設值] 按鈕，恢復到預設值。

6.0  錯誤訊息
---------------------

6.1  當 [LifeCam 檔案] 資料夾遺失時，就會出現「硬碟空間不足」錯誤

如果刪除 [我的文件] 內的 [LifeCam 檔案] 資料夾或無法使用該資料夾位置 (例如：位於網路共用區) 時，您的視訊剪輯就無法儲存，並可能收到錯誤訊息 :硬碟空間不足，無法擷取相片、音訊剪輯或視訊剪輯。請增加硬碟的可用空間，然後再試一次。 

若需修正此問題：
- 重新建立 [我的文件] 內的 [LifeCam 檔案] 資料夾。
  - 或 -
- 關閉 LifeCam 軟體，然後重新開啟。系統將會自動重新建立此資料夾。

---------------------

6.2  系統當機，並且 Tiptsf.dll 發生嚴重錯誤

您的電腦可能收到嚴重錯誤 (Tiptsf.dll 發生 STOP 錯誤)。 

若需修正此問題：
下載 Windows XP SP1 USB 1.1 修補程式。  
1.  造訪 http://support.microsoft.com。
2.  搜尋 KB827220。
3.  下載並安裝此修補程式。


---------------------

6.3  在 Windows XP SP1 中使用 LifeCam VX-6000：Usbport.sys 出現藍色畫面

您的電腦可能收到嚴重錯誤 (Usbport.sys 發生 STOP 錯誤)。 

若需修正此問題：
下載 Windows XP Service Pack 2 (SP2)。  
1.  造訪 http://update.microsoft.com。
2.  下載並安裝 Windows XP Service Pack 2 (SP2)。
