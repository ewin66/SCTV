1.1  Εάν έχετε λειτουργικό σύστημα Windows XP Service Pack 1 (SP1) θα πρέπει να κάνετε λήψη μιας ενημερωμένης έκδοσης (KB822603) 

Κατά την εγκατάσταση, ίσως εμφανιστεί το παρακάτω μήνυμα:
"To install the Microsoft LifeCam software, you must first install an update. 
You can download the update from the Microsoft Web site (KB822603)." (Για να εγκαταστήσετε το λογισμικό LifeCam θα πρέπει να κάνετε λήψη και να εγκαταστήσετε την ενημερωμένη έκδοση στον υπολογιστή σας. Μπορείτε να κάνετε λήψη της ενημερωμένης έκδοσης από την τοποθεσία Web της Microsoft (KB822603)).

ΣΗΜΕΙΩΣΗ: Εάν το μήνυμα ΔΕΝ εμφανιστεί, το πρόγραμμα εγκατάστασης διαπίστωσε ότι ΔΕΝ χρειάζεστε αυτή τη ενημερωμένη έκδοση ή ότι την έχετε ήδη.

Για την αποκατάσταση αυτού του σφάλματος:
1. Μεταβείτε στη διεύθυνση http://support.microsoft.com/support.
2. Αναζητήστε τον κωδικό KB822603.
3. Κάντε κλικ στη σύνδεση "Download the 822603 package now" (Λήψη του πακέτου 822603 τώρα).
4. Ακολουθήστε τις οδηγίες για λήψη και εγκατάσταση της ενημερωμένης έκδοσης.

Αφού εγκαταστήσετε την ενημερωμένη έκδοση, επανεκκινήστε το πρόγραμμα εγκατάστασης του λογισμικού LifeCam για να εγκαταστήσετε το λογισμικό LifeCam.

Αυτό το μήνυμα σφάλματος πιθανόν να εμφανιστεί για διάφορους λόγους:

- Ο σκληρός δίσκος όπου εγκαθιστάτε το λογισμικό ενδέχεται να είναι πλήρης.

1.3 Η εγκατάσταση μπορεί να φραγεί από προγράμμα προστασίας από ιούς, προγράμματα αποκλεισμού των αναδυόμενων παραθύρων ή προγράμματα ανίχνευσης λογισμικού υποκλοπής spyware

Κατά την εγκατάσταση του λογισμικού LifeCam, το πρόγραμμα προστασίας από τους ιούς, το πρόγραμμα αποκλεισμού αναδυόμενων παραθύρων ή το πρόγραμμα ανίχνευσης λογισμικού υποκλοπών ίσως
 αποκλείσει τη διαδικασία εγκατάστασης. 

Πολλά από αυτά τα προγράμματα θα σας ρωτήσουν εάν πρέπει να συνεχίσουν τον αποκλεισμό ή να επιτρέψουν την εγκατάσταση. 
Αν εμφανιστεί σχετικό μήνυμα προτροπής, επιτρέψτε στην εφαρμογή να εκτελέσει την εγκατάσταση. 

Για την αποκατάσταση αυτού του σφάλματος:
- Δοκιμάστε να κλείσετε προσωρινά το πρόγραμμα προστασίας από ιούς και να εκτελέσετε και πάλι το πρόγραμμα εγκατάστασης LifeCam.

Αφού ολοκληρωθεί η λήψη και η εγκατάσταση κάποιας Αυτόματης ενημέρωσης, θα πρέπει να κλείσετε το λογισμικό LifeCam και να το ανοίξετε

2.0  Εξοπλισμός
---------------------


2.1  Σχετικά με τις θύρες του Ενιαίου Σειριακού Διαύλου (Universal Serial Bus – USB)

Αν η κάμερα Web σας δεν λειτουργεί σωστά με τη θύρα USB στην οποία είναι συνδεδεμένη, δοκιμάστε να τη συνδέσετε σε άλλη θύρα USB.

Η Microsoft LifeCam λειτουργεί είτε συνδεθεί σε θύρα USB 1.1 είτε σε θύρα USB 2.0. 

Οι θύρες USB 1.1 και USB 2.0 είναι απολύτως όμοιες στην όψη.  Αν είναι δυνατόν, συνδέστε την κάμερα Web LifeCam σε θύρα USB 2.0.
Ορισμένες από τις δυνατότητες της κάμερας Web ίσως να μην υποστηρίζονται από θύρα USB 1.1.

Οι διάφορες θύρες USB ίσως επίσης παρέχουν διαφορετικά ποσά ισχύος στην κάμερα Web σας. Ένας διανομέας USB σε ένα πληκτρολόγιο, για παράδειγμα,
ίσως να μην τροφοδοτεί αρκετή ισχύ για την κάμερα Web LifeCam). Ίσως χρειαστεί να συνδέσετε την κάμερα Web σε μια θύρα USB στον ίδιο
τον υπολογιστή ή σε κάποιον τροφοδοτούμενο διανομέα USB.


---------------------

2.2 Η κάμερα Web πιθανώς να έχει χαμηλή απόδοση εάν στον υπολογιστή συνδεθεί υπερβολικός αριθμός συσκευών USB

Αν στον υπολογιστή σας είναι συνδεδεμένος υπερβολικός αριθμός συσκευών USB, η κάμερα Web LifeCam ίσως να μην λειτουργεί σωστά ή να πάψει να λειτουργεί. 

Στα συμπτώματα ενδέχεται να συγκαταλέγονται τα εξής: ο χώρος του βίντεο είναι κενός (δεν υπάρχει εικόνα ή ήχος), το Windows Live Messenger παύει να λειτουργεί (κρεμάει)
όταν αρχίζετε μια κλήση βίντεο ή η κάμερα Web δεν λειτουργεί στα βίντεο υψηλής ανάλυσης.

Για την αποκατάσταση αυτού του σφάλματος:
- Δοκιμάστε να αποσυνδέσετε και να επανασυνδέσετε την κάμερα web LifeCam.
- Δοκιμάστε να αποσυνδέσετε τις άλλες συσκευές USB που δεν χρησιμοποιούνται.
- Δοκιμάστε να χρησιμοποιήσετε άλλη θύρα USB του υπολογιστή σας.
- Εάν χρησιμοποιείτε διανομέα USB, προσπαθήστε να αποσυνδέσετε το διανομέα και να συνδέσετε την κάμερα web απευθείας στον υπολογιστή σας.

---------------------

2.3  Η LifeCam VX-6000 ίσως να μην λειτουργεί με την κάρτα ATI Radeon 9600

Όταν χρησιμοποιείτε την κάρτα βίντεο ATI Radeon 9600, η λήψη βίντεο από την LifeCam VX-6000 ενδεχομένως να παγώσει ή να "τρέμει". 

---------------------

2.4  Το βίντεο ίσως να εμφανίζεται σε πλήρη οθόνη

Ορισμένες κάρτες βίντεο διαθέτουν τη δυνατότητα να προβάλλουν βίντεο σε πλήρη οθόνη.

Αν προτιμάτε να μην προβάλετε βίντεο από τη LifeCam σε πλήρη οθόνη, συνήθως η δυνατότητα αυτή μπορεί να απενεργοποιηθεί.
Ανατρέξτε στην τεκμηρίωση του κατασκευαστή του υλικού για οδηγίες σχετικά με την απενεργοποίηση ή την κατάργηση αυτής της δυνατότητας.

---------------------

2.5  Το λογισμικό LifeCam ενδεχομένως να συντριβεί κατά την έναρξη και τη διακοπή λειτουργίας του βίντεο σε δεύτερη οθόνη

Αν χρησιμοποιείτε πολλαπλές οθόνες και εκτελείτε το λογισμικό LifeCam από δεύτερη οθόνη, το λογισμικό LifeCam ενδέχεται να συντριβεί.

Για την αποκατάσταση αυτού του σφάλματος:
 - Εκτελέστε το λογισμικό LifeCam στην κύρια οθόνη.

3.0  Λήψη βίντεο
---------------------

3.1  LifeCam VX-6000 -- όταν χρησιμοποιείται θύρα USB 1.1 δεν υποστηρίζονται όλες οι τιμές ανάλυσης

Όταν συνδεθεί σε θύρα USB 1.1, η LifeCam VX-6000 υποστηρίζει μέγιστη ανάλυση βίντεο 800 x 600 pixel
 και μέγιστη ανάλυση φωτογραφίας 1,9 megapixel. 

Για την αποκατάσταση αυτού του σφάλματος:
- Αν είναι δυνατόν, συνδέστε την κάμερα σε θύρα USB 2.0.
  - Ή -
- Μειώστε την ανάλυση.

---------------------

3.2  LifeCam VX-6000 -- Η ανάλυση βίντεο 1,3 MP δεν υποστηρίζει πανοραμική λήψη, κλίση, μεγέθυνση ή παρακολούθηση προσώπου

Αυτή τη στιγμή, οι δυνατότητες πανοραμικής λήψης, κλίσης, μεγέθυνσης και παρακολούθησης προσώπου δεν είναι διαθέσιμες για την ανάλυση βίντεο 1.3 MP (υψηλότερη).
Για να χρησιμοποιήσετε πανοραμική λήψη, κλίση, μεγέθυνση ή παρακολούθηση προσώπου, επιλέξτε χαμηλότερη ανάλυση βίντεο.

---------------------

3.3  LifeCam NX-6000 -- η λήψη προβολής ενδέχεται να εμφανιστεί κενή

Σε ορισμένες περιπτώσεις, η λήψη προβολής ενδέχεται να μην εμφανιστεί.

Για την αποκατάσταση αυτού του σφάλματος:
- Αποσυνδέστε την κάμερα web και συνδέστε την και πάλι.
  - Ή -
- Μειώστε την ανάλυση του βίντεο.

---------------------

3.4  Η οθόνη λήψης παραμένει στο προσκήνιο όταν χρησιμοποιείτε προγράμματα οδήγησης γενικής χρήσης

Σε ορισμένες περιπτώσεις, η λήψη βίντεο θα παραμείνει στο προσκήνιο, ακόμα και όταν ανοίξετε άλλα παράθυρα επάνω από τα παράθυρα του λογισμικού LifeCam.
Αυτό ενδέχεται να συμβεί όταν χρησιμοποιείτε προγράμματα οδήγησης γενικής χρήσης για την προβολή του βίντεο.

Για την αποκατάσταση αυτού του σφάλματος:
- Ανατρέξτε στην τεκμηρίωση του κατασκευαστή του υλικού για πληροφορίες σχετικά με την ενημέρωση των προγραμμάτων οδήγησης που χρησιμοποιείτε.


---------------------

3.5  Ορισμένα εφέ βίντεο ίσως προκαλέσουν επιβράδυνση της απόδοσης

Ορισμένα εφέ βίντεο ενδέχεται να μην αποδίδουν σωστά, όταν προστεθούν σε βίντεο υψηλής ανάλυσης.

Για τη βελτίωση της απόδοσης:
- Ρυθμίστε την ανάλυση του βίντεο σε χαμηλότερη τιμή.
  - Ή -
- Απενεργοποιήστε τα εφέ βίντεο.

4.0  Προγράμματα ανταλλαγής άμεσων μηνυμάτων
---------------------

4.1 Το κουμπί Windows Live Call της κάμερας Web δεν λειτουργεί

Σε ορισμένες περιπτώσεις, κατά τη διάρκεια της εγκατάσταση ή της ενημέρωσης του λογισμικού LifeCam, το κουμπί Windows Live Call
στην κάμερα web πιθανώς να μην έχει προετοιμαστεί σωστά. 

Για να λύσετε αυτό το πρόβλημα:

- Εκτελέστε αποσύνδεση και έπειτα επανσύνδεση στον υπολογιστή.
  - Ή - 
- Επανεκκινήστε τον υπολογιστή. 

---------------------

4.2  LifeCam NX-6000 -- Για ορισμένους πελάτες ανταλλαγής άμεσων μηνυμάτων ενδεχομένως να χρειάζεται το λογισμικό Microsoft LifeCam

Για να διασφαλιστεί η συμβατότητα των προϊόντων με ορισμένους πελάτες ανταλλαγής μηνυμάτων, όπως το Skype και το Yahoo! Messenger,
ενδεχομένως να απαιτείται το λογισμικό Microsoft LifeCam.

---------------------

4.3  Τα εφέ βίντεο δεν λειτουργούν σε όλα τα προγράμματα Ανταλλαγής άμεσων μηνυμάτων

Ορισμένα εφέ βίντεο δεν προβάλλονται σωστά στα προγράμματα ανταλλαγής άμεσων μηνυμάτων, όπως το Skype.

---------------------

4.4  Οι κλήσεις βίντεο στην Υπηρεσία ανταλλαγής άμεσων μηνυμάτων του America Online (AOL) ενδέχεται να αποσυνδεθούν

Αν προστατεύεστε από τείχος προστασίας, οι κλήσεις βίντεο που γίνονται με την Υπηρεσία ανταλλαγής άμεσων μηνυμάτων του AOL ενδέχεται να αποσυνδεθούν λίγο μετά την έναρξή τους.

Για περισσότερες πληροφορίες σχετικά με την προσαρμογή των ρυθμίσεων του τείχους προστασίας ή του δρομολογητή, βλ. www.aim.com.

5.0  Άλλες εφαρμογές
-----------------------

5.1  Η LifeCam VX-6000 δεν υποστηρίζεται από ορισμένα γνήσια προγράμματα 64 bit 

Η LifeCam VX-6000 δεν λειτουργεί με ορισμένα τοπικά προγράμματα 64 bit. Ωστόσο, στις περισσότερες περιπτώσεις, θα
λειτουργεί με τις εκδόσεις 32 bit αυτών των προγραμμάτων, σε περιβάλλον 64 bit.

---------------------

5.2 Η LifeCam VX-6000 ενδέχεται να μην εμφανίζεται στον Οδηγό Ρύθμισης ήχου και βίντεο Microsoft Office Communicator. 

Στον Microsoft Office Communicator, το LifeCam ρυθμίζεται μέσω του Οδηγού Ρύθμισης ήχου και βίντεο.
Σε ορισμένες περιπτώσεις, η LifeCam VX-6000 ενδεχομένως να μην εμφανίζεται στον κατάλογο των επιλογών.

Για την αποκατάσταση αυτού του σφάλματος:
-  Συνδέστε την κάμερα web LifeCam προτού ανάψετε τον υπολογιστή.

---------------------

5.3 Όταν κάνετε ανακοίνωση στα Windows Live Spaces, αντί για τον προσωπικό μου χώρο εμφανίζεται μια σελίδα ή κωδικός HTML όπου έχει ήδη γίνει σύνδεση ("Already signed in")

Εάν δημοσιεύσετε φωτογραφίες σε λογαριασμούς του Windows Live Spaces, χρησιμοποιώντας δύο λογαριασμούς χρήστη στον ίδιο υπολογιστή,
Windows Live Spaces θα εμφανίσει το σφάλμα "Already signed in" (ήδη συνδεδεμένος).

Για την αποκατάσταση αυτού του σφάλματος:
1.  Κλείστε όλα τα παράθυρα του προγράμματος περιήγησης Internet Explorer. 
2.  Στο λογισμικό LifeCam, κάντε κλικ και πάλι στο κουμπί "Post to my blog in Windows Live Spaces" (Ανακοίνωση στο λεύκωμά μου στα Windows Live Spaces) 

>>Για την πρόληψη εμφάνισης αυτού του σφάλματος:

- Να κλείνετε τον Internet Explorer όταν δεν τον χρησιμοποιείτε.

---------------------

5.4  Εάν κάποιος χρήστης αλλάξει την προεπιλεγμένη κάμερα web, η προεπιλεγμένη κάμερα web αλλάζει για όλους τους χρήστες.

Οι ρυθμίσεις του λογισμικού LifeCam ενδέχεται να επηρεάσουν όλα τα προφίλ χρήστη.  Για παράδειγμα, αν αλλάξετε την προεπιλεγμένη
κάμερα web σε ένα προφίλ χρήστη, η προεπιλεγμένη κάμερα web θα αλλάξει για όλα τα προφίλ χρήστη του συγκεκριμένου υπολογιστή.

6.0  Μηνύματα σφάλματος
---------------------

6.1  Όταν ο φάκελος LifeCam Files λείπει εμφανίζεται σφάλμα "Not enough hard disk space"

Αν διαγραφεί ο φάκελος LifeCam Files ή εάν η διεύθυνση του φακέλου πάψει να είναι διαθέσιμη
(για παράδειγμα, μέσω δικτύου κοινής χρήσης), τα βιντεοκλίπ σας δεν θα αποθηκευθούν και ίσως εμφανιστεί το μήνυμα σφάλματος
"There is not enough free hard disk space to capture a photo, audio clip, or video clip. Increase the amount
of free space on hard disk and then try again" (Δεν υπάρχει αρκετός ελεύθερος χώρος στο σκληρό δίσκο για να γίνει σύλληψη φωτογραφίας, κλιπ ήχου ή βιντεοκλίπ. Αυξήστε τον ελεύθερο χώρο στο σκληρό δίσκο και προσπαθήστε και πάλι). 

Για την αποκατάσταση αυτού του σφάλματος:
- Επαναδημιουργήστε το φάκελο "My Documents\LifeCam Files".
  - Ή -
- Κλείστε το λογισμικό LifeCam και ανοίξτε το και πάλι.  Ο φάκελος θα επαναδημιουργηθεί αυτόματα.

---------------------

6.2  Το σύστημα καταρρέει, με σοβαρό σφάλμα στο αρχείο Tiptsf.dll

Ο υπολογιστής σας ενδέχεται να εμφανίσει σοβαρό σφάλμα (STOP in Tiptsf.dll). 

Για την αποκατάσταση αυτού του σφάλματος:
Κάντε λήψη του λογισμικού ενημέρωσης κώδικα Windows XP SP1 USB 1.1.  
1.  Μεταβείτε στη διεύθυνση http://support.microsoft.com.
2.  Αναζητήστε τον κωδικό KB827220.
3.  Κάντε λήψη και εγκατάσταση του λογισμικού ενημέρωσης κώδικα.


---------------------

6.3  LifeCam VX-6000 με Windows XP SP1: Μπλε οθόνη με Usbport.sys

Ο υπολογιστής σας ενδέχεται να εμφανίσει σοβαρό σφάλμα (STOP in Usbport.sys). 

Για την αποκατάσταση αυτού του σφάλματος:
Κάντε λήψη των Windows XP Service Pack 2 (SP2).  
1.  Μεταβείτε στη διεύθυνση http://update.microsoft.com.
2.  Κάντε λήψη και εγκατάσταση των Windows XP Service Pack 2 (SP2).
