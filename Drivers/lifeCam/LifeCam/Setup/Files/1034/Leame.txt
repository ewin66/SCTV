1.0  Instalación y actualizaciones
---------------------

1.1  “Para instalar el software de Microsoft LifeCam, debe instalar primero una actualización”

Puede que reciba el siguiente mensaje durante la instalación:
“Para instalar el software de Microsoft LifeCam, debe instalar primero una actualización 
Puede descargar la actualización desde el sitio web de Microsoft (KB822603)".

NOTA: si no recibe este mensaje se debe a que el programa de instalación ha detectado que NO la necesita o que ya la tiene.

>>Para solucionar esto:
1. Vaya a http://support.microsoft.com/support.
2. Busque KB822603.
3. Haga clic en el vínculo "Descargue ahora el paquete 822603".
4. Siga las instrucciones para descargar e instalar la actualización.

Después de instalar la actualización, reinicie Instalar LifeCam para instalar el software de LifeCam. 

---------------------

1.2  "Mensaje de error de instalación"

Puede que reciba el siguiente mensaje de error de instalación con las versiones N y KN de Windows Vista:
"Error de instalación"

>>Para solucionar esto:
1. Vaya a http://support.microsoft.com/support.
2. Busque KB929399.
3. Haga clic en el vínculo para descargar el SDK del formato de Windows Media 11 en su equipo.
4. Siga las instrucciones para descargar e instalar la actualización.

Cuando finalice la instalación:
   Vaya al directorio C:\WMSDK\WMFSDK11\Redist e instale el programa del SDK en este directorio.


---------------------

1.3  Mensaje de error de instalación: no hay suficiente espacio en disco o se ha producido un error.

Puede recibir este mensaje de error por varios motivos:
- La protección antivirus, el bloqueador de elementos emergentes o los programas de detección de spyware pueden bloquear la instalación. (Si desea obtener más información, consulte la sección 1.4.)
- El disco duro donde realiza la instalación puede estar lleno.  

---------------------

1.4  La protección antivirus, el bloqueador de elementos emergentes o los programas de detección de spyware pueden bloquear la instalación.

Durante la instalación del software de LifeCam, la protección antivirus, el bloqueador de elementos emergentes o los programas de detección de spyware pueden bloquear la instalación. 

Muchos de estos programas le preguntarán si desea continuar con el bloqueo o permitir la instalación. 
Si éste es el caso, permita que el software se instale. 

>>Para solucionar esto:
- Intente salir temporalmente del programa de protección antivirus y vuelva a ejecutar el programa de instalación de LifeCam.

---------------------
 
1.5  Para completar la instalación de actualizaciones automáticas, cierre y vuelva a abrir el software de LifeCam
 
Después de haber descargado e instalado una actualización automática, debe cerrar el software de LifeCam y, a continuación, volver a abrirlo para que los cambios tengan efecto.

Para algunas actualizaciones, puede que se le solicite reiniciar el equipo para que los cambios tengan efecto.

---------------------

1.6  Después de instalar el software de la cámara web Logitech®, debe volver a instalar el software de Microsoft LifeCam (código 39 de mensaje de error).

El proceso de instalación del software de la cámara web Logitech puede causar problemas con el software de Microsoft LifeCam, incluido el código 39 de mensaje de error.

Para instalar el software de Microsoft LifeCam y Logitech QuickCam en el mismo equipo, instale los programas de Logitech QuickCam antes que los deMicrosoft LifeCam.  

Al usar Microsoft LifeCam, puede que tenga que cerrar el software de Logitech QuickCam o deshabilitarlo en la bandeja del sistema.

---------------------

1.7  Los controladores se eliminan después de desinstalar el software de LifeCam.

En Windows Vista, los controladores de Microsoft LifeCam se quitan después de desinstalar el software de Microsoft LifeCam.

>>Para volver a los controladores de LifeCam después de desinstalar el software:
- Conecte la cámara web. Aparecerá el asistente para agregar nuevo hardware.
- Haga clic en Examinar y navegue por el directorio de instalación (p. ej: C:\Archivos de programa\Microsoft LifeCam) para localizar y agregar los controladores.

2.0  Hardware
---------------------


2.1  Acerca de los puertos USB

Si la cámara web no funciona correctamente en el puerto USB usado, intente conectar la cámara web a otro puerto USB.

Microsoft LifeCam funcionará en un puerto USB 1.1 o USB 2.0. 

Los puertos USB 1.1 y USB 2.0 parecen idénticos.  Si es posible, use un puerto USB 2.0 para conectar la cámara web LifeCam. Puede que algunas de las características de la cámara web no sean compatibles con los puertos USB 1.1.

Los distintos puertos USB pueden proporcionar a la cámara web diferentes tipos de alimentación. Puede que un concentrador USB en el teclado no disponga de la alimentación suficiente para la cámara web LifeCam. Quizás necesite conectar la cámara web directamente al equipo o a un concentrador USB compatible con la alimentación de dispositivos.


---------------------

2.2  La cámara web puede tener un funcionamiento insuficiente o nulo si se conectan demasiados dispositivos USB al equipo

Si tiene demasiados dispositivos USB conectados al equipo, la cámara web LifeCam puede tener un rendimiento insuficiente o incluso dejar de funcionar. 

Entre los síntomas se puede incluir: el área de vídeo aparece en negro (sin imagen ni sonido), Windows Live Messenger deja de funcionar (se bloquea) al iniciar una videollamada, o puede que la cámara web no funcione con vídeos de alta resolución.

>>Para solucionar esto:
- Intente desconectar y volver a conectar la cámara web LifeCam.
- Intente desconectar otros dispositivos USB no usados.
- Intente usar otro puerto USB del equipo.
- Si usa un concentrador USB, desconecte el concentrador y conecte la cámara web directamente al equipo.

---------------------

2.3  Puede que la LifeCam VX-6000 no funcione con la tarjeta ATI Radeon 9600

Al usar la tarjeta de vídeo ATI Radeon 9600, la captura de vídeo de la LifeCam VX-6000 puede congelarse o parpadear. 

---------------------

2.4  Puede que el vídeo se vea como una imagen en pantalla completa

Algunas tarjetas de vídeo incluyen características para mostrar vídeo como una imagen en pantalla completa.

Si prefiere no ver el vídeo de LifeCam en modo de pantalla completa, normalmente puede deshabilitar esta característica.  Consulte la documentación del fabricante del hardware para obtener instrucciones acerca de cómo deshabilitar o desactivar esta característica.

---------------------

2.5  Puede que LifeCam se bloquee al iniciar o detener el vídeo en un monitor secundario

Si usa varios monitores y ejecuta LifeCam desde un monitor secundario, puede que se bloquee el software de LifeCam.

>>Para evitar este error:
 - Ejecute el software de LifeCam en el monitor principal.

3.0  Captura de vídeo
---------------------

3.1  El puerto USB 1.1 no es compatible con todas las resoluciones.

Conectados a un puerto USB 1.1, los productos LifeCam admiten una resolución máxima de vídeo de 800 x 600 píxeles y una resolución fotográfica máxima de 1,9 megapíxeles. 

>>Para solucionar esto:
- Si puede, use un puerto USB 2.0.
  - O bien -
- Elija una resolución de vídeo menor.

---------------------

3.2  La máxima resolución de vídeo no es compatible con las opciones de desplazar, inclinar, zoom y de rastreo facial

Al usar las máximas resoluciones de la cámara web, las características de desplazar, inclinar, zoom y de rastreo facial no se encuentran disponibles. Para usar las características de desplazar, inclinar, zoom y de rastreo facial, elija una resolución de vídeo menor.

---------------------

3.3  LifeCam NX-6000 -- la captura de previsualización puede aparecer en negro

En algunos casos, puede que no aparezca la captura de previsualización.

>>Para solucionar esto:
- Desconecte la cámara web y vuelva a conectarla.
  - O bien -
- Elija una resolución de vídeo menor.

---------------------

3.4  La pantalla de captura se mantiene en un segundo plano al usar los controladores de vídeo genéricos

En algunos casos, la captura de vídeo se mantendrá en un segundo plano aunque abra otras ventanas por encima del software de LifeCam.
Esto puede suceder si usa controladores genéricos para su adaptador de vídeo.

>>Para solucionar esto:
- Consulte la documentación del fabricante del hardware para obtener información sobre cómo actualizar los controladores de vídeo.

4.0  Programas de mensajería instantánea
---------------------

4.1  Puede que el software de Microsoft LifeCam sea necesario para determinados clientes de mensajería

Para garantizar la compatibilidad con algunos clientes de mensajería, como Skype y Yahoo! Messenger, puede que necesite el software de Microsoft LifeCam.

---------------------

4.2  Los efectos de vídeo no funcionan en todos los programas de mensajería instantánea

Algunos efectos de vídeo no se mostrarán en programas de mensajería instantánea, como Skype.

---------------------

4.3  Las videollamadas de America Online (AOL) Instant Messenger pueden desconectarse

Si está detrás de un firewall, las llamadas de vídeo establecidas con AOL Instant Messenger pueden desconectarse poco después de iniciarse.

>> Si desea obtener más información acerca del ajuste para firewall o enrutador, consulte www.aim.com.


---------------------

4.4  El micrófono no está disponible en las videollamadas con Windows Live Messenger.

Puede que, con Windows Live Messenger, el volumen del micrófono esté deshabilitado y no se pueda ajustar en las versiones N y KN de Windows Vista.

>>Para solucionar esto:
1. Vaya a http://support.microsoft.com/support.
2. Busque KB929399.
3. Haga clic en el vínculo para descargar el SDK del formato de Windows Media 11 en su equipo.
4. Siga las instrucciones para descargar e instalar la actualización.

Cuando finalice la instalación:
   Vaya al directorio C:\WMSDK\WMFSDK11\Redist e instale el programa del SDK en este directorio.

5.0  Otras aplicaciones
-----------------------

5.1  Puede que LifeCam VX-6000 no aparezca en el Asistente para ajuste de audio y vídeo de Microsoft Office Communicator. 

En Microsoft Office Communicator, LifeCam se configura mediante el Asistente para ajuste de audio y vídeo.  En algunos casos, puede que LifeCam VX-6000 no aparezca en la lista de opciones.

>>Para solucionar esto:
-  Conecte la cámara web LifeCam antes de encender el equipo.

---------------------

5.2  Al publicar en Windows Live Spaces, aparece la página "Ya se ha iniciado la sesión" o código HTML, en lugar de mi espacio personal

Si envía fotografías a cuentas de Windows Live Spaces con dos cuentas de usuario en el mismo equipo, Windows Live Spaces producirá un error del tipo "Ya se ha iniciado la sesión".

>>Para solucionar esto:
1.  Cierre todas las ventanas de Internet Explorer. 
2.  En el software de LifeCam, vuelva a hacer clic en el botón "Enviar a mi blog en Windows Live Spaces". 

>>Para evitar este error:

- Cierre Internet Explorer entre las diferentes sesiones de usuario.

---------------------

5.3  Si un usuario cambia la cámara web predeterminada, ésta cambia para todos los usuarios.

Los ajustes del software de LifeCam pueden afectar a todos los perfiles de usuario.  Por ejemplo, si cambia la cámara web predeterminada en un perfil de usuario, ésta cambiará para todos los perfiles de usuario de ese equipo.

6.0  Mensajes de error
---------------------

6.1  Error "No hay suficiente espacio disponible en el disco" si falta la carpeta Archivos de LifeCam

Si elimina la carpeta "Mis documentos\Archivos de LifeCam" o si la ubicación de la carpeta no se encuentra disponible (p. ej. en un recurso compartido de red), no se guardarán los clips de video y puede que reciba el siguiente mensaje de error: "No hay suficiente espacio disponible en el disco duro para capturar una foto, clip de audio o clip de vídeo. Aumente la cantidad de espacio en el disco duro e inténtelo de nuevo." 

>>Para solucionar esto:
- Vuelva a crear la carpeta "Mis documentos\Archivos de LifeCam".
  - O bien -
- Cierre el software de LifeCam y vuelva a abrirlo.  La carpeta se volverá a crear automáticamente.

---------------------

6.2  El sistema se bloquea, con error grave en Tiptsf.dll

El equipo puede recibir un error grave (STOP en Tiptsf.dll). 

>>>Para solucionar esto:
Descargue Windows XP Service Pack 2 (SP2).  
1.  Vaya a http://update.microsoft.com.
2.  Descargue e instale Windows XP Service Pack 2 (SP2).


---------------------

6.3  LifeCam VX-6000: pantalla azul en Usbport.sys

El equipo puede recibir un error grave (STOP en Usbport.sys). 

>>>Para solucionar esto:
Descargue Windows XP Service Pack 2 (SP2).  
1.  Vaya a http://update.microsoft.com.
2.  Descargue e instale Windows XP Service Pack 2 (SP2).


---------------------

6.4  El software de LifeCam se bloquea al abrir el software de Nero Video Decoder

Esto puede suceder al instalar determinadas versiones de Nero Video Decoder.

Para solucionar este problema, actualice o desinstale el software.  

- Consulte la documentación del fabricante del hardware para obtener información acerca de cómo actualizar el software.


---------------------

6.5  El software de LifeCam se bloquea al abrir el software. Pegasus PICVideo

Esto puede suceder al instalar determinadas versiones de Pegasus PICVideo.

Para solucionar este problema, desinstale Pegasus PICVideo.

Puede consultar al fabricante si hay actualizaciones disponibles.
