1.0  Instalação e atualizações
---------------------

1.1  "To install the Microsoft LifeCam software, you must first install an update." (Para instalar o software da Microsoft LifeCam, primeiro você deve instalar uma atualização.)

Você pode receber a seguinte mensagem durante a instalação:
"To install the Microsoft LifeCam software, you must first install an update." (Para instalar o software da Microsoft LifeCam, primeiro você deve instalar uma atualização.) 
"You can download the update from the Microsoft Web site." (Você pode baixar a atualização do site da Microsoft.) — (KB822603)

OBSERVAÇÃO: Se você NÃO recebeu a mensagem, a instalação detectou que você NÃO precisa dessa atualização ou ela já está instalada.

>>Para corrigir isso:
1. Acesse http://support.microsoft.com/support.
2. Procure KB822603.
3. Clique no link "Download the 822603 package now" (Baixar agora o pacote 822603).
4. Siga as instruções para baixar e instalar a atualização.

Depois de instalar a atualização, reinicie a Instalação da LifeCam para instalar o software da LifeCam. 

---------------------

1.2  "Erro de instalação"

No Windows Vista versões N e KN, você pode receber a seguinte mensagem durante a instalação:
"Erro de instalação"

>>Para corrigir isso:
1. Acesse http://support.microsoft.com/support.
2. Procure KB929399.
3. Clique no link para baixar o Windows Media Format 11 SDK em seu computador.
4. Siga as instruções para baixar e instalar a atualização.

Depois de concluir esta instalação:
   Vá até o diretório C:\WMSDK\WMFSDK11\Redist e instale o programa SDK nesse diretório.


---------------------

1.3  Mensagem de erro de instalação: Espaço em disco insuficiente ou ocorreu um erro.

Você pode receber esta mensagem de erro por vários motivos:
– A instalação pode ter sido bloqueada pela proteção antivírus, por um bloqueador de pop-up ou por programas de detecção de spyware. Para obter mais informações, consulte 1.4.
– O disco rígido em que você está instalando o software pode estar cheio.  

---------------------

1.4  A instalação pode ter sido bloqueada pela proteção antivírus, bloqueador de pop-up ou programas de detecção de spyware

Durante a instalação do software da LifeCam, é possível que um programa antivírus, bloqueador de pop-up ou de detecção de spyware bloqueie o processo de instalação. 

Muitos desses programas perguntarão se você deseja continuar bloqueando ou se permite a instalação. 
Se for solicitado, permita que o software seja instalado. 

>>Para corrigir isso:
– Tente fechar temporariamente o programa antivírus e execute novamente a instalação da LifeCam.

---------------------
 
1.5  Para concluir a instalação da Atualização Automática, feche o software da LifeCam e abra-o novamente
 
Depois de baixar e instalar uma atualização automática, é necessário fechar o software da LifeCam e abri-lo novamente para que as alterações sejam implementadas.

Em algumas atualizações, você poderá ser solicitado também a reinicializar o computador para que as alterações sejam implementadas.

---------------------

1.6  Após a instalação do software da webcam Logitech®, o software da Microsoft LifeCam deverá ser reinstalado (mensagem de erro Código 39)

O processo de instalação do software da webcam Logitech pode causar problemas com o software da Microsoft LifeCam, inclusive a mensagem de erro Código 39.

Para instalar o software da Microsoft LifeCam e o software Logitech QuickCam no mesmo computador, instale o software Logitech QuickCam software antes de instalar o software da Microsoft LifeCam.  

Para usar a Microsoft LifeCam, talvez também seja necessário sair do software Logitech QuickCam ou desativá-lo na bandeja do sistema.

---------------------

1.7  Após a desinstalação do software da LifeCam, os drivers são removidos

No Windows Vista, após a desinstalação do software da Microsoft LifeCam, os drivers da Microsoft LifeCam são removidos.

>>Para restaurar os drivers da LifeCam após a desinstalação do software:
– Conecte a webcam. Será exibido o Assistente para Adicionar Novo Hardware.
– Clique em Procurar e navegue até o diretório de instalação (por exemplo: C:\Arquivos de Programas\Microsoft LifeCam) para localizar e adicionar os drivers.

2.0  Hardware
---------------------


2.1  Sobre as portas USB (Universal Serial Bus)

Se a webcam não funcionar corretamente na porta USB à qual está conectada, tente conectá-la a uma porta USB diferente.

A Microsoft LifeCam funcionará tanto em uma porta USB 1.1 quanto em uma porta USB 2.0. 

As portas USB 1.1 e USB 2.0 parecem idênticas.  Se possível, use uma porta USB 2.0 para conectar sua webcam LifeCam. Alguns recursos da webcam talvez não tenham suporte em portas USB 1.1.

Portas USB diferentes podem fornecer quantidades de energia diferentes à webcam. Por exemplo, um hub USB em um teclado pode não fornecer energia suficiente para a webcam LifeCam. Talvez você precise conectar a webcam a uma porta USB no computador ou em um hub USB ligado.


---------------------

2.2  A webcam poderá ter um desempenho insatisfatório ou nulo se houver muitos dispositivos USB conectados ao computador

Se houver muitos dispositivos USB conectados ao computador, a webcam LifeCam poderá apresentar desempenho insatisfatório ou parar de funcionar. 

Os sintomas podem incluir: a área do vídeo fica preta (sem imagens nem som), o Windows Live Messenger pára de funcionar (trava) quando você inicia uma chamada de vídeo, ou a webcam não funciona com vídeos de alta resolução.

>>Para corrigir isso:
– Tente desconectar e reconectar a webcam LifeCam.
– Tente desconectar outros dispositivos USB não utilizados.
– Tente usar outra porta USB no computador.
– Se estiver usando um hub USB, tente desconectá-lo e conectar a webcam diretamente ao computador.

---------------------

2.3  A LifeCam VX-6000 pode não funcionar com a placa ATI Radeon 9600

Se você estiver usando a placa de vídeo ATI Radeon 9600, a captura de vídeo da LifeCam VX-6000 poderá congelar ou piscar. 

---------------------

2.4  O vídeo pode exibir uma imagem de tela inteira

Algumas placas de vídeo incluem recursos para exibir vídeo como imagens de tela inteira.

Se você preferir não ver o vídeo da LifeCam no modo de tela inteira, normalmente esse recurso poderá ser desabilitado.  Consulte a documentação do fabricante do hardware para obter instruções sobre como desabilitar ou desligar esse recurso.

---------------------

2.5  A LifeCam trava na inicialização ou na parada de vídeo em um segundo monitor

Se você usar vários monitores e executar a LifeCam em um monitor secundário, o software da LifeCam poderá travar.

>>Para impedir esse erro:
 – Execute o software da LifeCam no monitor principal.

3.0  Captura de vídeo
---------------------

3.1  Nem todas as resoluções têm suporte em uma porta USB 1.1

Quando conectada a uma porta USB 1.1, a LifeCam VX-6000 oferece suporte à resolução de vídeo máxima de 800 x 600 pixels, e à resolução de foto máxima de 1.9 megapixels. 

>>Para corrigir isso:
– Se houver disponibilidade, conecte-a a uma porta USB 2.0.
  – Ou –
– Reduza a resolução.

---------------------

3.2  Não há suporte aos recursos de panorâmica, inclinação, zoom e rastreio de face quando é usada a resolução máxima de vídeo

Ao ajustar a resolução máxima para a webcam, os recursos de panorâmica, inclinação, zoom e rastreio de face não ficarão disponíveis. Para usar esses recursos, escolha uma resolução de vídeo menor.

---------------------

3.3  LifeCam NX-6000 -- a captura de visualização pode aparecer preta

Em alguns casos, a captura de visualização pode não aparecer.

>>Para corrigir isso:
– Desconecte e reconecte a webcam.
  – Ou –
– Reduza a resolução de vídeo.

---------------------

3.4  A tela de captura permanece em primeiro plano quando são usados drivers de vídeo genéricos

Em alguns casos, a captura de vídeo permanece em primeiro plano mesmo que você abra outras janelas sobre o software da LifeCam.
Isso pode acontecer quando são usados drivers genéricos para exibição de vídeo.

>>Para corrigir isso:
– Consulte a documentação do fabricante do hardware para obter informações sobre como atualizar os drivers de vídeo.

4.0  Programas de mensagens instantâneas
---------------------

4.1  O software da Microsoft LifeCam pode ser necessário para determinados clientes de mensagens

Para garantir a compatibilidade do produto com alguns clientes de mensagens, como o Skype e Yahoo! Messenger, pode ser necessário o software da Microsoft LifeCam.

---------------------

4.2  Os efeitos de vídeo não funcionam em todos os programas de mensagens instantâneas

Alguns efeitos de vídeo não são exibidos em programas de mensagens instantâneas, como o Skype.

---------------------

4.3  As chamadas de vídeo no America Online (AOL) Instant Messenger podem ser desconectadas

Se você estiver protegido por um firewall, as chamadas de vídeo realizadas com o AOL Instant Messenger poderão ser desconectadas logo após terem sido iniciadas.

>> Para obter mais informações sobre o ajuste das configurações de firewall ou de roteador, consulte www.aim.com.


---------------------

4.4  Microfone não disponível nas chamadas de vídeo realizadas no Windows Live Messenger.

No Windows Vista versões N e KN, o volume do microfone pode ficar desabilitado e não ajustável ao usar o Windows Live Messenger.

>>Para corrigir isso:
1. Acesse http://support.microsoft.com/support.
2. Procure KB929399.
3. Clique no link para baixar o Windows Media Format 11 SDK em seu computador.
4. Siga as instruções para baixar e instalar a atualização.

Depois de concluir esta instalação:
   Vá até o diretório C:\WMSDK\WMFSDK11\Redist e instale o programa SDK nesse diretório.

5.0  Outros aplicativos
-----------------------

5.1  A LifeCam VX-6000 pode não aparecer no Assistente para Ajuste de Áudio e Vídeo do Microsoft Office Communicator. 

No Microsoft Office Communicator, a LifeCam é configurada com o Assistente para Ajuste de Áudio e Vídeo.  Em alguns casos, a LifeCam VX-6000 pode não aparecer na lista de opções.

>>Para corrigir isso:
– Conecte a webcam LifeCam antes de ligar o computador.

---------------------

5.2  Ao postar no Windows Live Spaces, vejo a página "Already signed in" (Já conectado) ou um código HTML em vez do meu espaço pessoal

Se você postar fotos em contas do Windows Live Spaces usando duas contas de usuário no mesmo computador, o Windows Live Spaces retornará a mensagem de erro "Already signed in" (Já conectado).

>>Para corrigir isso:
1.  Feche todas as janelas do navegador Internet Explorer. 
2.  No software da LifeCam, clique novamente no botão "Post to my blog in Windows Live Spaces" (Postar no meu blog no Windows Live Spaces). 

>>Para impedir esse erro:

– Feche o Internet Explorer entre as sessões de usuário.

---------------------

5.3  Se um usuário alterar a webcam padrão, ela será alterada para todos os usuários.

As configurações do software da LifeCam podem afetar todos os perfis de usuários.  Por exemplo, se você alterar a webcam padrão em um perfil de usuário, ela será alterada para todos os perfis de usuário nesse computador.

6.0  Mensagens de erro
---------------------

6.1  Erro "Espaço insuficiente no disco rígido" quando a pasta Arquivos da LifeCam está ausente

Se a pasta "Meus Documentos\Arquivos da LifeCam" for excluída, ou se o local da pasta ficar indisponível (por exemplo, em um compartilhamento de rede), seus clipes de vídeo não serão salvos e você poderá receber a seguinte mensagem de erro: "There is not enough free hard disk space to capture a photo, audio clip, or video clip. Increase the amount of free space on hard disk and then try again" (Não há espaço livre suficiente no disco rígido para capturar uma foto, clipe de áudio ou clipe de vídeo. Aumente a quantidade de espaço livre no disco rígido e tente novamente.) 

>>Para corrigir isso:
– Recrie a pasta "Meus Documentos\Arquivos da LifeCam".
  – Ou –
– Feche o software da LifeCam e abra-o novamente.  A pasta será automaticamente recriada.

---------------------

6.2  O sistema trava, com um erro grave em Tiptsf.dll

O computador pode receber uma mensagem de erro grave (STOP em Tiptsf.dll). 

>>>Para corrigir isso:
Baixe o Service Pack 2 (SP2) do Windows XP.  
1.  Acesse http://update.microsoft.com.
2.  Baixe e instale o Service Pack 2 (SP2) do Windows XP.


---------------------

6.3  LifeCam VX-6000: Tela azul em Usbport.sys

O computador pode receber uma mensagem de erro grave (STOP em Usbport.sys). 

>>>Para corrigir isso:
Baixe o Service Pack 2 (SP2) do Windows XP.  
1.  Acesse http://update.microsoft.com.
2.  Baixe e instale o Service Pack 2 (SP2) do Windows XP.


---------------------

6.4  O software da LifeCam trava ao abrir o software Nero Video Decoder

Isso pode ocorrer quando determinadas versões do Nero Video Decoder estão instaladas.

Para corrigir o problema, atualize ou desinstale o software.  

Consulte a documentação do fabricante do hardware para obter informações sobre como atualizar o software.


---------------------

6.5  O software da LifeCam trava ao abrir o software Pegasus PICVideo

Isso pode ocorrer quando determinadas versões do Pegasus PICVideo estão instaladas.

Para resolver o problema, desinstale o Pegasus PICVideo.

Você pode também consultar o fabricante para saber se há atualizações disponíveis.
