1.0  安裝與更新
---------------------

1.1  「若要安裝 Microsoft LifeCam 軟體，必須先安裝更新程式」

在進行安裝時，您可能會收到以下訊息：
「若要安裝 Microsoft LifeCam 軟體，必須先安裝更新。 
您可從 Microsoft 網站下載更新檔 (KB822603)」。

注意：如果未收到此訊息，這是因為安裝程式偵測到您並不需要或已經安裝此更新程式。

>>若要修正此錯誤：
1. 請造訪 http://support.microsoft.com/support。
2. 搜尋 KB822603。
3. 按一下 [Download the 822603 package now](立即下載 822603 套件) 連結。
4. 請依照指示下載並安裝更新程式。

安裝更新程式之後，請重新啟動 LifeCam 安裝程式，以便安裝 LifeCam 軟體。 

---------------------

1.2  「安裝錯誤」

在 Windows Vista N 與 KN 版本中，您可能會在安裝過程中收到下列訊息：
「安裝錯誤」

>>若要修正此錯誤：
1. 請造訪 http://support.microsoft.com/support。
2. 搜尋 KB929399。
3. 按一下連結，為您的電腦下載 Windows Media Format 11 SDK。
4. 請依照指示下載並安裝更新程式。

完成此安裝後：
   移至 C:\WMSDK\WMFSDK11\Redist 目錄，並在此目錄中安裝 SDK 程式。


---------------------

1.3  安裝錯誤訊息：硬碟空間不足或發生錯誤。

收到此錯誤訊息的原因有很多種：
- 防毒程式、快顯封鎖程式或間諜軟體偵測程式可能會導致安裝被封鎖。 (若需更多資訊，請參閱 1.4。)
- 您要安裝軟體的硬碟機可能已滿。  

---------------------

1.4  防毒程式、快顯封鎖程式或間諜軟體偵測程式可能會導致安裝被封鎖

在安裝 LifeCam 軟體期間，您的防毒程式、快顯封鎖程式或間諜軟體偵測程式可能封鎖軟體安裝程序。 

但是大部分的程式都會詢問您是否允許或不允許繼續安裝。 
出現提示時，請允許程式繼續安裝。 

>>若要修正此錯誤：
- 請嘗試暫時結束防毒程式，然後重新執行 LifeCam 安裝程式。

---------------------
 
1.5  若要完成自動更新安裝，請關閉 LifeCam 軟體後再重新開啟
 
下載並安裝自動更新後，您必須關閉 LifeCam 軟體然後再度開啟，變更才會生效。

某些更新程式可能還會提示您重新啟動電腦，以使變更生效。

---------------------

1.6  安裝 Logitech® 網路攝影機軟體後，必須重新安裝 Microsoft LifeCam 軟體 (錯誤訊息代碼 39)

Logitech 網路攝影機軟體安裝程序可能會導致 Microsoft LifeCam 軟體發生問題，包括顯示錯誤訊息代碼 39。

若要在同一台電腦上安裝 Microsoft LifeCam 軟體和 Logitech QuickCam 軟體，請先安裝 Logitech QuickCam 軟體，然後再安裝 Microsoft LifeCam 軟體。  

使用 Microsoft LifeCam 時，您可能還需要關閉或從系統匣停用 Logitech QuickCam 軟體。

---------------------

1.7  解除安裝 LifeCam 軟體之後，驅動程式也會一併被移除。

在 Windows Vista 中，解除安裝 Microsoft LifeCam 軟體之後，Microsoft LifeCam 的驅動程式也會一併被移除。

>>若要在解除安裝軟體之後還原 LifeCam 驅動程式：
- 插入網路攝影機。 [新增硬體精靈] 將會出現。
- 按一下 [瀏覽] 並瀏覽至安裝目錄 (例如：C:\Program Files\Microsoft LifeCam)，找到並新增驅動程式。

2.0  硬體
---------------------


2.1  關於通用序列匯流排 (USB) 連接埠

如果網路攝影機在目前所使用的 USB 連接埠上無法正常運作，請嘗試將網路攝影機插入其他 USB 連接埠。

Microsoft LifeCam 可以在 USB 1.1 或 USB 2.0 連接埠上使用。 

USB 1.1 與 USB 2.0 連接埠的外觀相同。  可能的話，請使用 USB 2.0 連接埠來連接 LifeCam 網路攝影機。 USB 1.1 連接埠可能不支援某些網路攝影機功能。

不同的 USB 連接埠可能會為網路攝影機提供不同程度的電力。 例如，鍵盤上的 USB 集線器可能無法為 LifeCam 網路攝影機提供足夠的電力。 您可能需要將網路攝影機連接到電腦主機上的 USB 連接埠，或已連接電源的 USB 集線器上。


---------------------

2.2 若連接至電腦的 USB 裝置過多，可能會導致網路攝影機執行不當或無法執行

若您連接至電腦的 USB 裝置過多，LifeCam 網路攝影機可能會執行不當或停止運作。 

問題可能包括：視訊區變黑 (沒有畫面或聲音)、啟動視訊通話後 Windows Live Messenger 會停止運作 (當機)，或網路攝影機在高解析度視訊時無法運作。

>>若要修正此錯誤：
- 嘗試拔除後重新連接 LifeCam 網路攝影機。
- 嘗試拔除其他閒置的 USB 裝置。
- 嘗試使用電腦上的其他 USB 連接埠。
- 如果是使用 USB 集線器，請嘗試拔下集線器，然後直接將網路攝影機插入電腦中。

---------------------

2.3  LifeCam VX-6000 可能無法搭配 ATI Radeon 9600 視訊卡使用

使用 ATI Radeon 9600 視訊卡時，從 LifeCam VX-6000 上擷取的視訊可能會定格或閃爍。 

---------------------

2.4  視訊可能顯示成全螢幕影像

部分視訊卡具有將視訊顯示成全螢幕影像的功能。

如果不想用全螢幕模式觀看 LifeCam 視訊，通常可停用此功能。  請參閱硬體製造商文件，以取得有關停用或關閉此功能的指示。

---------------------

2.5  在次要電腦螢幕上開始或停止視訊時，LifeCam 可能當機

如果使用多台電腦螢幕，並從次要電腦螢幕上執行 LifeCam 時，LifeCam 軟體可能會當機。

>>若要避免此錯誤：
 - 在主要電腦螢幕上執行 LifeCam 軟體。

3.0  視訊擷取
---------------------

3.1  當使用 USB 1.1 連接埠時，並不支援所有的解析度

當連接到 USB 1.1 連接埠時，LifeCam 產品支援的最高視訊解析度為 800 x 600 像素，最高相片解析度為 1.9 MP (190 萬像素)。 

>>若要修正此錯誤：
- 如果有的話，請連接至 USB 2.0 連接埠。
  - 或 -
- 降低解析度。

---------------------

3.2  使用最高解析度時，視訊解析度不支援畫面上下左右移動、縮放與臉部追蹤等功能

若為網路攝影機設定最高解析度，則無法使用畫面上下左右移動、縮放與臉部追蹤功能。 若要使用畫面上下左右移動、縮放與臉部追蹤等功能，請選擇較低的視訊解析度。

---------------------

3.3  LifeCam NX-6000 -- 預覽擷取畫面可能呈現黑色

某些情況下，預覽擷取畫面可能無法顯示。

>>若要修正此錯誤：
- 先拔下然後再插入網路攝影機。
  - 或 -
- 降低視訊解析度。

---------------------

3.4  使用一般視訊驅動程式時，擷取畫面會保持在前景顯示狀態

在某些情況下，即使在 LifeCam 軟體上開啟其他視窗，視訊擷取畫面仍會保持在前景中。
這可能是因為您的視訊顯示是使用一般驅動程式。

>>若要修正此錯誤：
- 請參閱硬體製造商文件，以取得有關如何更新視訊驅動程式的資訊。

4.0  即時訊息程式
---------------------

4.1  某些訊息用戶端可能需要安裝 Microsoft LifeCam 軟體

為確保產品與某些訊息用戶端相容 (如 Skype 和 Yahoo!  Messenger)，可能需要使用 Microsoft LifeCam 軟體。

---------------------

某些視訊特效無法在即時訊息程式中顯示，如 Skype。

---------------------

如果您的電腦位於防火牆內，使用 AOL Instant Messenger 建立的視訊通話可能會在啟動後立即中斷。

>> 若需更多有關調整防火牆或路由器設定的資訊，請參閱 www.aim.com。


---------------------

1. 請造訪 http://support.microsoft.com/support。
2. 搜尋 KB929399。
3. 按一下連結，為您的電腦下載 Windows Media Format 11 SDK。
4. 請依照指示下載並安裝更新程式。

完成此安裝後：
   移至 C:\WMSDK\WMFSDK11\Redist 目錄，並在此目錄中安裝 SDK 程式。

5.0  其他應用程式
-----------------------

5.1  Microsoft Office Communicator 的 [音訊和視訊調整精靈] 中可能不會顯示 LifeCam VX-6000。 

在 Microsoft Office Communicator 中，可透過 [音訊和視訊調整精靈] 設定 LifeCam。  在某些情況下，LifeCam VX-6000 可能不會顯示在選單中。

>>若要修正此錯誤：
-  先插上 LifeCam 網路攝影機之後再開啟電腦。

---------------------

5.2 張貼到 Windows Live Spaces 時，出現的是 [已經登入] 頁面或 HTML 程式碼，而非我的個人分享空間

在同一台電腦上，如果同時使用兩個使用者帳戶向 Windows Live Spaces 中張貼照片，Windows Live Spaces 就會出現「已經登入」錯誤。

>>若要修正此錯誤：
1.  請關閉所有 Internet Explorer 瀏覽器視窗。 
2.  在 LifeCam 軟體上，再按一下 [張貼到我的 Windows Live Spaces 部落格] 按鈕。 

>>若要避免此錯誤：

請關閉在使用者工作階段中的 Internet Explorer。

---------------------

5.3  如果某位使用者變更了預設網路攝影機，則所有使用者的預設攝影機也會隨之變更。

LifeCam 軟體設定可能會影響所有的使用者設定檔。 比如說，如果您變更了一個使用者設定檔中的預設網路攝影機，則該電腦上所有使用者設定檔的預設網路攝影機都會隨之變更。

6.0  錯誤訊息
---------------------

6.1  當 [LifeCam 檔案] 資料夾遺失時，就會出現「硬碟空間不足」錯誤

如果刪除 [我的文件\LifeCam 檔案] 資料夾，或是無法使用該資料夾位置 (例如：位於網路共用區) 時，您的視訊剪輯就不會被儲存，同時您可能會收到錯誤訊息 [硬碟空間不足，無法擷取相片、音訊剪輯或視訊剪輯。 請增加硬碟的可用空間，然後再試一次。] 

>>若要修正此錯誤：
- 重新建立 [我的文件\LifeCam 檔案] 資料夾。
  - 或 -
- 關閉 LifeCam 軟體，然後重新開啟。 系統將會自動重新建立此資料夾。

---------------------

6.2  Tiptsf.dll 發生嚴重錯誤並且導致系統當機

您的電腦可能收到嚴重錯誤 (Tiptsf.dll 發生 STOP 錯誤)。  

>>>若要修正此錯誤：
下載 Windows XP Service Pack 2 (SP2)。  
1.  造訪 http://update.microsoft.com。
2.  下載並安裝 Windows XP Service Pack 2 (SP2)。


---------------------

6.3  LifeCam VX-6000：Usbport.sys 出現藍色畫面

您的電腦可能收到嚴重錯誤 (Usbport.sys 發生 STOP 錯誤)。  

>>>若要修正此錯誤：
下載 Windows XP Service Pack 2 (SP2)。  
1.  造訪 http://update.microsoft.com。
2.  下載並安裝 Windows XP Service Pack 2 (SP2)。


---------------------

6.4  開啟軟體 (Nero Video Decoder) 時，LifeCam 軟體當機

安裝某些版本的 Nero Video Decoder 時，可能會發生此問題。

更新或解除安裝軟體可解決此問題。  

請參閱硬體製造商文件，以取得有關如何更新軟體的資訊。


---------------------

6.5  開啟軟體時，LifeCam 軟體當機。 (Pegasus PICVideo)

安裝某些版本的 Pegasus PICVideo 時，可能會發生此問題。

解除安裝 Pegasus PICVideo 可解決此問題。

您也可以向製造商諮詢，瞭解是否有可用更新。
