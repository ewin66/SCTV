1.0  インストールおよび更新
---------------------

1.1  「Microsoft LifeCam ソフトウェアをインストールするには、まず初めに更新プログラムをインストールする必要があります。」

セットアップ時に次のメッセージが表示される場合があります。
「Microsoft LifeCam ソフトウェアをインストールするには、まず初めに更新プログラムをインストールする必要があります。
更新プログラムは、Microsoft Web サイトからダウンロードできます (KB822603)。」

メモ: メッセージが表示されない場合は、この更新プログラムが不要であるか、すでに適用済みです。

>>この問題を解決するには
1. http://support.microsoft.com/support にアクセスします。
2. KB822603 を検索します。
3.「Windows XP SP1 USB 1.1 および 2.0 更新プログラムについて」のサポート技術情報の中にある [822603 パッケージ] リンクをクリックして、[ダウンロード] をクリックします。
4. 更新プログラムのインストール指示に従います。

更新プログラムをインストールしたら、LifeCam セットアップを再起動して、LifeCam ソフトウェアをインストールします。

---------------------

1.2  「インストール エラー」

Windows Vista N および KN バージョンでは、セットアップ時に次のメッセージが表示される場合があります。
「インストール エラー」

>>この問題を解決するには
1. http://support.microsoft.com/support にアクセスします。
2. KB929399 を検索します。
3. リンクをクリックし、お使いのコンピュータ用の Windows Media Format 11 SDK をダウンロードします。
4. 更新プログラムのインストール指示に従います。

このインストールの完了後:
   C:¥WMSDK¥WMFSDK11¥Redist ディレクトリに移動し、このディレクトリの SDK プログラムをインストールします。


---------------------

1.3  「ディスクの空き領域が不足している」というインストール エラー メッセージが表示される、またはエラーが発生する

このエラー メッセージが表示される原因はいくつかあります。
- ウィルス対策プログラム、ポップアップ ブロック ソフトウェア、またはスパイウェア検出プログラムにより、インストールが中断されている (詳細については、1.4 を参照してください)。
- ソフトウェアのインストール先ハード ディスクに空き領域がない

---------------------

1.4  ウィルス対策プログラム、ポップアップ ブロック ソフトウェア、またはスパイウェア検出プログラムにより、インストールが中断されることがある

LifeCam ソフトウェアのインストール中、ウィルス対策プログラム、ポップアップ ブロック ソフトウェア、またはスパイウェア検出プログラムによってインストール処理が中断されることがあります。

これらのプログラムの多くでは、中断したままにするか、インストールを許可するかを尋ねるメッセージが表示されます。
このようなメッセージが表示された場合は、ソフトウェアのインストールを許可してください。

>>この問題を解決するには
- ウィルス対策プログラムを一時的に停止して、LifeCam のセットアップをもう一度実行します。

---------------------

1.5  自動更新インストールを完了するには、LifeCam ソフトウェアをいったん閉じてから開き直す必要がある

自動更新プログラムをダウンロードしてインストールした後は、変更を適用するために、LifeCam ソフトウェアをいったん閉じてから開き直す必要があります。

一部の更新プログラムでは、変更を適用するために、コンピュータの再起動も必要になる場合があります。

---------------------

1.6  Logicool® Web カメラ ソフトウェアをインストールした後で、Microsoft LifeCam ソフトウェアをインストールする必要がある (コード 39 エラー メッセージ)

Logicool Web カメラ ソフトウェアのインストール処理によって、Microsoft LifeCam ソフトウェアに問題が発生する場合があります (コード 39 エラー メッセージなど)。

Microsoft LifeCam ソフトウェアと Logicool QuickCam ソフトウェアの両方を 1 台のコンピュータにインストールするには、先に Logicool QuickCam ソフトウェアをインストールしてから、Microsoft LifeCam ソフトウェアをインストールしてください。

また、Microsoft LifeCam の使用中は、Logicool QuickCam ソフトウェアを終了するか、システム トレイで無効にする必要のある場合があります。

2.0  ハードウェア
---------------------


2.1  USB (Universal Serial Bus) ポートについて

接続している USB ポートで Web カメラが正しく動作しない場合は、別の USB ポートに Web カメラを差し込んでください。

Microsoft LifeCam は、USB 1.1 または USB 2.0 ポートで動作します。

USB 1.1 ポートと USB 2.0 ポートの外観は同じです。可能な場合は、USB 2.0 ポートに LifeCam Web カメラを接続してください。一部の Web カメラ機能は、USB 1.1 ポートで使用できないことがあります。

USB ポートごとに Web カメラに供給される電力が異なることに注意してください。たとえば、キーボード上の USB ハブは、LifeCam Web カメラに十分な電力を供給できないことがあります。この場合は、コンピュータの USB ポート、またはセルフパワーの USB ハブに Web カメラを差し込んでください。


---------------------

2.2 コンピュータに接続されている USB デバイスが多すぎる場合は、Web カメラが適切に動作しなかったり、まったく動作しないことがある

コンピュータに接続されている USB デバイスが多すぎる場合は、LifeCam Web カメラが適切に動作しなかったり、動作しなくなることがあります。

この問題には次のような症状があります: ビデオ領域が黒くなる (画像、音声ともなし)、ビデオ通話を開始すると Windows Live Messenger が動作しなくなる (ハングする)、または Web カメラが高解像度で動作しなくなる

>>この問題を解決するには
- LifeCam Web カメラを取り外して接続し直します。
- 使用していない他の USB デバイスを取り外します。
- コンピュータに付属している別の USB ポートを使用します。
- USB ハブを使用している場合は、ハブを取り外し、Web カメラをコンピュータに直接接続します。

---------------------

2.3  LifeCam VX-6000 は ATI Radeon 9600 カードとの組み合わせで正しく使用できないことがある

ATI Radeon 9600 ビデオ カードを使用している場合、LifeCam VX-6000 でキャプチャしたビデオを再生すると、ビデオの再生が止まったり、画面がちらつくことがあります。

---------------------

2.4  ビデオが全画面表示されることがある

一部のビデオ カードには、ビデオを全画面で表示する機能があります。

LifeCam ビデオを全画面表示モードで表示しないようにするために、通常、ビデオ カードのこの機能を無効にすることができます。この機能を無効にする方法については、ハードウェアの製造元のマニュアルを参照してください。

---------------------

2.5  プライマリ モニタ以外でビデオの再生を開始および停止すると、LifeCam がクラッシュすることがある

複数のモニタを使用している場合にプライマリ モニタ以外から LifeCam を実行すると、LifeCam ソフトウェアがクラッシュすることがあります。

>> このエラーを回避するには
 - LifeCam ソフトウェアはプライマリ モニタから実行してください。





---------------------
3.0  ビデオ キャプチャ
---------------------

3.1  USB 1.1 ポートを使用する場合は、一部の解像度がサポートされない

USB 1.1 ポートに接続した場合、LifeCam 製品では最大で 800 x 600 ピクセルのビデオ、および最大で 1.9 メガピクセルの写真がサポートされます。

>>この問題を解決するには
- 可能な場合は、USB 2.0 ポートに接続します。
  - または -
- 解像度を低くします。

---------------------

3.2  ビデオ解像度を最高にすると、パン、チルト、ズーム、フェイス トラッキング機能がサポートされない

Web カメラのビデオ解像度を最高に設定すると、パン、チルト、ズーム、およびフェイス トラッキング機能を利用できません。パン、チルト、ズーム、またはフェイス トラッキングを使用するには、ビデオ解像度を低くしてください。

---------------------

3.3  LifeCam NX-6000 -- プレビュー キャプチャが黒く表示されることがある

プレビュー キャプチャが表示されないことがあります。

>>この問題を解決するには
- Web カメラを取り外してから接続し直します。
  - または -
- ビデオ解像度を低くします。

---------------------

3.4  一般的なビデオ ドライバの使用時に、キャプチャ画面が最前面に表示されたままになる

LifeCam ソフトウェアの上に他のウィンドウを開いても、ビデオ キャプチャ画面が最前面に表示されたままになることがあります。
この問題は、ビデオ表示用に一般的なドライバを使用している場合に発生することがあります。

>>この問題を解決するには
- ハードウェアの製造元のマニュアルを参照して、ビデオ ドライバを更新してください。

4.0  インスタント メッセージング プログラム
---------------------

4.1  一部のメッセージング クライアントで、Microsoft LifeCam ソフトウェアが必要になることがある

Skype や Yahoo!メッセンジャーなど、一部のメッセージング クライアントとの互換性を保つために、Microsoft LifeCam ソフトウェアが必要になることがあります。

---------------------

4.2  一部のビデオ効果がすべてのインスタント メッセージング プログラムで機能しない

一部のビデオ効果は、Skype など一部のインスタント メッセージング プログラムで機能しません。

---------------------

4.3  America Online (AOL) インスタント メッセンジャーでのビデオ通話が切断されることがある

コンピュータがファイアウォールの内側にある場合、AOL インスタント メッセンジャーでビデオ通話を開始すると、すぐに切断されることがあります。

>> ファイアウォールまたはルーター設定の調整については、www.aim.com を参照してください。


---------------------

4.4  Windows Live Messenger のビデオ通話でマイクを利用できない

Windows Vista N および KN バージョンでは、Windows Live Messenger の使用時に、マイクが無効になったり、音量調整できないことがあります。

>>この問題を解決するには
1. http://support.microsoft.com/support にアクセスします。
2. KB929399 を検索します。
3. リンクをクリックし、お使いのコンピュータ用の Windows Media Format 11 SDK をダウンロードします。
4. 更新プログラムのインストール指示に従います。

このインストールの完了後:
   C:¥WMSDK¥WMFSDK11¥Redist ディレクトリに移動し、このディレクトリの SDK プログラムをインストールします。

5.0  その他のアプリケーション
-----------------------

5.1  LifeCam VX-6000 が Microsoft Office Communicator のオーディオとビデオのチューニング ウィザードに表示されないことがある

Microsoft Office Communicator では、LifeCam をオーディオとビデオのチューニング ウィザードでセットアップします。このとき、選択肢の一覧に LifeCam VX-6000 が表示されないことがあります。

>>この問題を解決するには
-  コンピュータの電源を入れる前に、LifeCam Web カメラを接続します。

---------------------

5.2 Windows Live スペースへの投稿時に、「既にサインインしています」ページが表示される、または個人用スペースではなく HTML コードが表示される

1 つのコンピュータで、2 つのユーザー アカウントを使用して Windows Live スペース アカウントに写真を投稿すると、「既にサインインしています」というエラー メッセージが表示されます。

>>この問題を解決するには
1.  すべての Internet Explorer ブラウザ ウィンドウを閉じます。
2.  LifeCam ソフトウェアで、[Windows Live スペースのマイ ブログに投稿] ボタンをもう一度クリックします。

>>このエラーを回避するには

- ユーザー アカウントを切り替えるたびに Internet Explorer を閉じます。

---------------------

5.3  あるユーザーが既定の Web カメラを変更すると、すべてのユーザーの既定の Web カメラが変更される

LifeCam ソフトウェア設定はすべてのユーザー プロファイルに影響することがあります。たとえば、あるユーザー プロファイルの既定の Web カメラを変更した場合、既定の Web カメラはそのコンピュータ上のすべてのユーザー プロファイルで変更されます。

6.0  エラー メッセージ
---------------------

6.1  [LifeCam ファイル] フォルダが存在しない場合に、「ハード ディスクの空き領域が不足している」というエラーが表示される

[マイ ドキュメント] フォルダ内の [LifeCam ファイル] フォルダを削除したり、フォルダの場所が使用できなくなった場合 (ネットワーク上の共有フォルダなど)、ビデオ クリップが保存されず、エラー メッセージ「写真、オーディオ クリップ、またはビデオ クリップのキャプチャに必要なハード ディスクの空き領域が足りません。ハード ディスクの空き領域を増やしてから、やり直してください。」が表示されることがあります。

>>この問題を解決するには
- [マイ ドキュメント] フォルダに [LifeCam ファイル] フォルダを再作成します。
  - または -
- LifeCam ソフトウェアをいったん閉じてから開き直します。これでフォルダが自動的に再作成されます。

---------------------

6.2  Tiptsf.dll の重大なエラーでシステムがクラッシュする

お使いのコンピュータで、重大なエラーが発生する場合があります (Tiptsf.dll で停止)。

>>>この問題を解決するには
Windows XP Service Pack 2 (SP2) をダウンロードします。
1.  http://update.microsoft.com にアクセスします。
2.  Windows XP Service Pack 2 (SP2) をダウンロードしてインストールします。


---------------------

6.3  LifeCam VX-6000: Usbport.sys でブルー スクリーンが表示される

お使いのコンピュータで、重大なエラーが発生する場合があります (Usbport.dll で停止)。

>>>この問題を解決するには
Windows XP Service Pack 2 (SP2) をダウンロードします。
1.  http://update.microsoft.com にアクセスします。
2.  Windows XP Service Pack 2 (SP2) をダウンロードしてインストールします。


---------------------

6.4  Nero Video Decoder ソフトウェアを起動すると LifeCam ソフトウェアがクラッシュする

この問題は、特定のバージョンの Nero Video Decoder がインストールされている場合に発生します。

問題を解決するには、Nero Video Decoder ソフトウェアを更新またはアンインストールします。

ハードウェアの製造元のマニュアルを参照して、Nero Video Decoder ソフトウェアを更新してください。


---------------------

6.5  Pegasus PICVideo ソフトウェアを起動すると LifeCam ソフトウェアがクラッシュする

この問題は、特定のバージョンの Pegasus PICVideo がインストールされている場合に発生します。

問題を解決するには、Pegasus PICVideo ソフトウェアをアンインストールします。

または、更新プログラムがあるかどうかを製造元に確認してください。
