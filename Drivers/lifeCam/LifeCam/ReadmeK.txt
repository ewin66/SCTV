1.0  설치 및 업데이트
---------------------

1.1  Windows XP 서비스 팩 1(SP1)을 실행하는 경우 업데이트(KB822603)를 다운로드해야 합니다.

설치 도중 다음과 같은 메시지가 표시될 수 있습니다.
"Microsoft LifeCam 소프트웨어를 설치하려면 먼저 업데이트를 설치해야 합니다. 
업데이트는 Microsoft 웹 사이트에서 다운로드할 수 있습니다(KB822603)."

참고: 설치 프로그램에서 이 업데이트가 필요하지 않음을 발견했거나 이미 업데이트가 설치된 경우 이 메시지가 표시되지 않습니다.

이 문제를 수정하려면:
1. http://support.microsoft.com/support를 방문합니다.
2. KB822603을 검색합니다.
3. "지금 822603 패키지 다운로드" 링크를 클릭합니다.
4. 지침을 따라 업데이트를 다운로드하여 설치합니다.

업데이트를 설치한 다음 LifeCam 설치 프로그램을 다시 시작하여 LifeCam 소프트웨어를 설치하십시오. 

---------------------

1.2  디스크 공간 부족 또는 오류 발생 등 설치 관련 오류 메시지가 나타납니다.

이러한 오류 메시지가 표시되는 원인은 여러 가지입니다.
- 바이러스 방지, 팝업 차단 또는 스파이웨어 탐지 프로그램으로 인해 설치가 차단될 수 있습니다(자세한 내용은 1.3을 참조하십시오).
- 소프트웨어를 설치하는 하드 디스크가 꽉 찼을 수 있습니다.  

---------------------

1.3  바이러스 방지, 팝업 차단 또는 스파이웨어 검색 프로그램으로 인해 설치가 차단될 수 있습니다.

LifeCam 소프트웨어를 설치하는 도중 바이러스 방지 프로그램, 팝업 차단 또는 스파이웨어 검색 프로그램 등 여러 가지 원인으로 인해
설치 프로세스가 차단될 수 있습니다. 

대부분의 프로그램에서는 설치를 차단하거나 허용할지를 묻는 메시지를 사용자에게 표시하므로, 
메시지가 표시되면 소프트웨어 설치를 허용하십시오. 

이 문제를 수정하려면:
- 바이러스 방지 프로그램을 일시적으로 종료한 다음 LifeCam 설치 프로그램을 다시 실행해 보십시오.

---------------------
 
1.4  자동 업데이트 설치를 완료하려면 LifeCam 소프트웨어를 닫은 다음 다시 엽니다.
 
자동 업데이트를 다운로드하여 설치한 다음 LifeCam 소프트웨어를 닫고 다시
시작해야 변경 사항이 적용됩니다.

일부 업데이트는 변경 사항을 적용하기 위해 컴퓨터를 다시 시작하라는 메시지가 표시될 수도 있습니다.

---------------------

1.5  Logitech® 웹캠 소프트웨어를 설치한 다음 Microsoft LifeCam 소프트웨어를 다시 설치해야 합니다(Code 39 오류 메시지).

Logitech 웹캠 설치 프로세스는 Microsoft LifeCam 소프트웨어와 문제를 일으키며,
Code 39 오류 메시지가 이러한 문제에 포함됩니다.

Microsoft LifeCam 소프트웨어와 Logitech QuickCam 소프트웨어를 같은 컴퓨터에 설치하려면,
Logitech QuickCam 소프트웨어를 먼저 설치한 다음 Microsoft LifeCam 소프트웨어를 설치합니다.  

Microsoft LifeCam을 사용할 때에는 Logitech QuickCam 소프트웨어를 종료하거나 시스템 트레이에서 사용하지 않도록 설정해야 합니다.

2.0  하드웨어
---------------------


2.1  USB(범용 직렬 버스) 포트 정보

웹캠이 연결되어 있는 USB 포트와 올바르게 작동하지 않는 경우 웹캠을 다른 USB 포트에 연결해 봅니다.

Microsoft LifeCam은 USB 1.1 또는 USB 2.0 포트와 모두 작동합니다. 

USB 1.1과 USB 2.0 포트는 모양이 똑같습니다.  가능하면 USB 2.0 포트를 사용하여 LifeCam 웹캠을 연결하십시오.
USB 1.1 포트에서는 웹캠의 일부 기능이 지원되지 않을 수 있습니다.

USB 포트에 따라 웹캠에 공급되는 전력량이 다를 수도 있습니다. 예를 들어 키보드에 있는 USB 허브에서
공급되는 전력은 LifeCam 웹캠에 충분하지 않을 수 있습니다. 컴퓨터 본체에 있는 USB 포트 또는
전원이 공급되는 USB 허브에 웹캠을 연결해야 할 수 있습니다.


---------------------

2.2  컴퓨터에 너무 많은 USB 장치를 연결할 경우 웹캠 성능이 저하되거나 작동하지 않을 수 있습니다.

컴퓨터에 너무 많은 USB 장치를 연결하면 LifeCam 웹캠 작동이 불안정해지거나 작동을 멈출 수 있습니다. 

다음과 같은 증상이 나타날 수 있습니다. 화상 통화를 시작할 때 비디오 영역이 검정으로 표시되거나(화면 또는 소리 없음) 
Windows Live Messenger 작동이 중지되거나(정지), 웹캠이 높은 해상도에서 작동하지 않습니다.

이 문제를 수정하려면:
- LifeCam 웹캠을 연결 해제한 다음 다시 연결해 봅니다.
- 다른 USB 장치를 연결 해제해 봅니다.
- 컴퓨터의 다른 USB 포트에 연결해 봅니다.
- USB 허브를 사용하는 경우 허브를 연결 해제하고 웹캠을 컴퓨터에 직접 연결해 봅니다.

---------------------

2.3  LifeCam VX-6000이 ATI Radeon 9600 카드와 작동하지 않을 수 있습니다.

ATI Radeon 9600 비디오 카드를 사용하는 경우 LifeCam VX-6000의 비디오 캡처가 멈추거나 깜박거릴 수 있습니다. 

---------------------

2.4  비디오가 전제 화면 이미지로 표시될 수 있습니다.

일부 비디오 카드에는 비디오를 전체 화면 이미지로 표시하는 기능이 포함되어 있습니다.

LifeCam 비디오를 전체 화면 모드로 보지 않으려면 이 기능을 일반적으로 사용하지 않도록 설정할 수 있습니다.
이 기능을 사용하지 않도록 설정하거나 끄는 방법은 하드웨어 제조업체 설명서를 참조하십시오.

---------------------

2.5  보조 모니터에서 비디오를 시작하거나 정지하면 LifeCam이 충돌할 수 있습니다.

다중 모니터를 사용하는 경우 LifeCam을 보조 모니터에서 실행하면 LifeCam 소프트웨어가 충돌할 수 있습니다.

이 오류를 방지하려면:
 - LifeCam 소프트웨어를 기본 모니터에서 실행합니다.

3.0  비디오 캡처
---------------------

3.1  LifeCam VX-6000 -- USB 1.1 포트를 사용하는 경우 일부 해상도가 지원되지 않습니다.

USB 1.1 포트에 연결할 경우 LifeCam VX-6000에서 지원하는 최대 비디오 해상도는 800 x 600픽셀이고
최대 사진 해상도는 1.9메가픽셀입니다. 

이 문제를 수정하려면:
- 사용 가능한 경우, USB 2.0 포트에 연결합니다.
  또는
- 해상도를 낮춥니다.

---------------------

3.2  LifeCam VX-6000 -- 1.3메가픽셀 비디오 해상도에서 왼쪽/오른쪽으로 이동, 위쪽/아래쪽으로 이동, 확대/축소 및 페이스 트래킹이 지원되지 않습니다.

왼쪽/오른쪽으로 이동, 위쪽/아래쪽으로 이동, 확대/축소 및 페이스 트래킹 기능은 현재 1.3메가픽셀(최고) 비디오 해상도에서는 사용할 수 없습니다.
이러한 기능을 사용하려면 낮은 비디오 해상도를 선택하십시오.

---------------------

3.3  LifeCam NX-6000 -- 캡처 미리보기가 검정으로 표시될 수 있습니다.

경우에 따라 캡처 미리보기가 표시되지 않습니다.

이 문제를 수정하려면:
- 웹캠을 연결 해제한 다음 다시 연결합니다.
  또는
- 비디오 해상도를 낮춥니다.

---------------------

3.4  일반 비디오 드라이버를 사용할 경우 캡처 화면이 전경에 계속 표시됩니다.

경우에 따라 LifeCam 소프트웨어 위에 다른 창을 열더라도 비디오 캡처가 전경에 계속 표시됩니다.
비디오 디스플레이에 일반 드라이버를 사용할 경우 이런 현상이 나타날 수 있습니다.

이 문제를 수정하려면:
- 하드웨어 제조업체 설명서에서 비디오 드라이버 업데이트 방법을 참조합니다.


---------------------

3.5  일부 비디오 효과가 성능을 저하시킬 수 있습니다.

일부 비디오 효과를 고해상도 비디오에 추가하면 성능을 저하시킬 수 있습니다.

성능을 개선하려면:
- 비디오 해상도를 더 낮은 값으로 설정합니다.
  또는
- 비디오 효과를 끕니다.

4.0  인스턴트 메시징 프로그램
---------------------

4.1  웹캠의 Windows Live Call 단추가 작동하지 않습니다.

경우에 따라 LifeCam 소프트웨어를 설치하거나 업데이트하는 도중 웹캠의 Windows Live Call 단추가
올바르게 초기화되지 않을 수 있습니다. 

이 문제를 해결하려면:

- 컴퓨터에서 로그오프한 다음 다시 로그인합니다.
  또는 
- 컴퓨터를 다시 시작합니다. 

---------------------

4.2  LifeCam NX-6000 -- 메시징 클라이언트에 따라 Microsoft LifeCam 소프트웨어가 필요할 수 있습니다.

Skype 및 Yahoo! Messenger와 같은 일부 메시징 클라이언트와의 제품 호환성을 유지하려면
Microsoft LifeCam 소프트웨어가 필요할 수 있습니다.

---------------------

4.3  일부 인스턴트 메신저 프로그램에서는 비디오 효과가 작동하지 않습니다.

Skype와 같은 인스턴트 메신저 프로그램에서 일부 비디오 효과가 표시되지 않습니다.

---------------------

4.4  AOL(America Online) 인스턴트 메신저에서 화상 전화가 연결되지 않습니다.

방화벽을 사용하는 경우 AOL 인스턴트 메신저를 통한 화상 전화가 시작 직후 연결 해제될 수 있습니다.

방화벽 또는 라우터 설정을 조정하는 자세한 내용은 www.aim.com을 참조하십시오.

5.0  기타 응용 프로그램
-----------------------

5.1  일부 기본 64비트 프로그램은 LifeCam VX-6000을 지원하지 않습니다. 

LifeCam VX-6000은 일부 기본 64비트 프로그램과 작동하지 않습니다. 하지만 대부분의 경우
64비트 운영 체제에서 실행되는 32비트 버전의 프로그램과는 작동합니다.

---------------------

5.2  LifeCam VX-6000이 Microsoft Office Communicator의 오디오 및 비디오 조정 마법사에 표시되지 않습니다. 

Microsoft Office Communicator에서 LifeCam은 오디오 및 비디오 조정 마법사를 통해 설정됩니다.
경우에 따라 LifeCam VX-6000이 선택 목록에 표시되지 않을 수 있습니다.

이 문제를 수정하려면:
-  LifeCam 웹캠을 연결한 다음 컴퓨터를 켭니다.

---------------------

5.3  Windows Live Spaces에 게시할 때 개인 스페이스 대신 "이미 로그인하였습니다" 페이지 또는 HTML 코드가 표시됩니다.

한 컴퓨터에서 두 개의 사용자 계정을 사용하여 Windows Live Spaces에 사진을 게시하면,
Windows Live Spaces에서 "이미 로그인하였습니다" 오류가 표시됩니다.

이 문제를 수정하려면:
1.  모든 Internet Explorer 브라우저 창을 닫습니다. 
2.  LifeCam 소프트웨어에서 "내 Windows Live Spaces 블로그에 게시" 단추를 다시 한 번 클릭합니다. 

>>이 오류를 방지하려면:

- 사용자 세션을 전환할 때 Internet Explorer를 닫습니다.

---------------------

5.4  한 사용자가 기본 웹캠을 변경하면 모든 사용자의 기본 웹캠이 변경됩니다.

LifeCam 소프트웨어 설정은 모든 사용자 프로필에 영향을 줄 수 있습니다.  예를 들어 한 사용자 프로필에서 기본
웹캠이 변경되면 해당 컴퓨터의 모든 사용자 프로필에 대한 기본 웹캠도 변경됩니다.


5.5 특정 언어에서 LifeCam 속성 창이 완전히 열리지 않습니다.

Windows XP 중국어 및 한국어 버전에서는 Microsoft Lifecam 속성 창의 일부가 표시되지 않습니다. 
이로 인해 설정을 원래 값으로 복원하는 데 사용되는 기본값 단추가 일부분만 표시됩니다. 
이러한 보기와 상관없이 기본값 단추는 올바르게 작동합니다.

6.0  오류 메시지
---------------------

6.1  LifeCam 파일 폴더가 없는 경우 "하드 디스크 공간 부족" 오류가 발생할 수 있습니다.

"My Documents\LifeCam Files" 폴더가 삭제되거나 폴더 위치를 사용할 수 없는 경우(예:
네트워크 공유) 비디오 클립이 저장되지 않고 
"사진, 오디오 클립 또는 비디오 클립 캡처에 사용할 하드 디스크 공간이 충분하지 않습니다. 하드 디스크의
사용 가능한 공간을 늘린 다음 다시 시도하십시오."라는 오류 메시지가 표시될 수 있습니다. 

이 문제를 수정하려면:
- "My Documents\LifeCam Files" 폴더를 다시 만듭니다.
  또는
- LifeCam 소프트웨어를 닫은 다음 다시 엽니다.  폴더가 자동으로 다시 만들어집니다.

6.2  Tiptsf.dll의 심각한 오류로 인해 시스템이 충돌합니다.

컴퓨터에서 심각한 오류(Tiptsf.dll 중지)가 발생할 수 있습니다. 

이 문제를 수정하려면:
Windows XP SP1 USB 1.1 패치를 다운로드합니다.  
1.  http://support.microsoft.com을 방문합니다.
2.  KB827220을 검색합니다.
3.  패치를 다운로드하여 설치합니다.

6.3  Windows XP SP1에 설치된 LifeCam VX-6000에서 Usbport.sys 관련 블루 스크린이 표시됩니다.

컴퓨터에서 심각한 오류(Usbport.sys 중지)가 발생할 수 있습니다. 

이 문제를 수정하려면:
Windows XP 서비스 팩 2(SP2)를 다운로드합니다.  
1.  http://update.microsoft.com을 방문합니다.
2.  Windows XP 서비스 팩 2(SP2)를 다운로드하여 설치합니다.
