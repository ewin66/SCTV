1.0  安装和更新
---------------------

1.1  如果运行 Windows XP Service Pack 1 (SP1) 则必须下载更新 (KB822603)

安装过程中，您可能收到下列消息：
“要安装 Microsoft LifeCam 软件，必须首先安装更新。 
您可以从 Microsoft 网站下载更新 (KB822603)。”

注意：如果您未收到该消息，说明安装程序已检测到您不需要此更新，或者您已经具有此更新。

要解决此错误，请按照下列步骤执行操作：
1. 转至 http://support.microsoft.com/support。
2. 搜索 KB822603。
3. 单击“立即下载 822603 程序包”链接。
4. 按照说明下载并安装更新。

安装更新之后，重新启动 LifeCam 安装程序以安装 LifeCam 软件。 

---------------------

1.2  安装错误消息：磁盘空间不足或发生了错误。

您收到此错误消息可能有多种原因：
- 防病毒保护、弹出窗口阻止程序或间谍软件检测程序会阻止安装。（有关详细信息，请参阅 1.3。）
- 您正在安装软件的硬盘可能已满。  

---------------------

1.3  防病毒保护、弹出窗口阻止程序或间谍软件检测程序会阻止安装

在安装 LifeCam 软件期间，防病毒保护程序、弹出窗口阻止程序或间谍软件检测程序可能会
阻止安装过程。 

其中许多程序会询问您是继续阻止还是允许安装。 
如果出现提示，请允许安装软件。 

要解决此错误，请按照下列步骤执行操作：
- 尝试暂时退出防病毒保护程序并再次运行 LifeCam 安装程序。

---------------------
 
1.4  要完成自动更新安装，请关闭然后再次打开 LifeCam 软件
 
下载并安装自动更新之后，必须关闭然后再次打开 LifeCam 软件，
所做的更改才会生效。

对于某些更新，系统可能提示您重新启动计算机之后更改才会生效。

---------------------

1.5  安装 Logitech® 网络摄像机软件之后，必须重新安装 Microsoft LifeCam 软件（代码 39 错误消息）

Logitech 网络摄像机软件安装过程可能导致 Microsoft LifeCam 软件出现问题，
包括代码 39 错误消息。

要在同一台计算机上同时安装 Microsoft LifeCam 软件和 Logitech QuickCam 软件，
请在安装 Microsoft LifeCam 软件之前安装 Logitech QuickCam 软件。  

当使用 Microsoft LifeCam 时，您还可能需要关闭 Logitech QuickCam 软件或从系统任务栏中禁用它。

2.0  硬件
---------------------


2.1  关于通用串行总线 (USB) 端口

如果网络摄像机使用所连接的 USB 端口无法正常运行，请尝试将网络摄像机插入另一个 USB 端口。

您的 Microsoft LifeCam 将使用 USB 1.1 或 USB 2.0 端口。 

USB 1.1 和 USB 2.0 端口看似相同。如果可能，使用 USB 2.0 端口连接 LifeCam 网络摄像机。
USB 1.1 端口可能不支持网络摄像机的某些功能。

不同的 USB 端口还可能为网络摄像机提供不同的电量。 例如，键盘上的 USB 集线器
可能不能为 LifeCam 网络摄像机提供足够的电量。 您可能需要将网络摄像机插入计算机上的 USB 端口，
或者插入通电的 USB 集线器。


---------------------

2.2  如果计算机连接的 USB 设备太多，网络摄像机可能运行状态较差或根本不运行

如果计算机连接的 USB 设备太多，LifeCam 网络摄像机可能运行状态较差或停止运行。 

症状可能包括：视频区域呈现黑色（无图片或声音）；当您开始视频通话时，Windows Live Messenger 停止工作（挂断）；
或者网络摄像机在较高分辨率下不工作。

要解决此错误，请按照下列步骤执行操作：
- 尝试断开并重新连接 LifeCam 网络摄像机。
- 尝试断开其他没有使用的 USB 设备。
- 尝试使用计算机上的其他 USB 端口。
- 如果使用 USB 集线器，请尝试拔出集线器，并将网络摄像机直接插接到计算机。

---------------------

2.3  LifeCam VX-6000 使用 ATI Radeon 9600 卡时可能不工作

如果使用 ATI Radeon 9600 视频卡，LifeCam VX-6000 捕获视频时可能冻结或闪烁。 

---------------------

2.4  视频可能显示为全屏图像

某些视频卡包括将视频显示为全屏图像的功能。

如果您不喜欢在全屏模式下观看 LifeCam 视频，则通常可禁用此功能。
有关禁用或关闭此功能的说明，请参阅硬件制造商文档。

---------------------

2.5  当在辅助监视器上启动和停止视频时，LifeCam 可能崩溃

如果您使用多台监视器，并且从辅助监视器运行 LifeCam，则 LifeCam 软件可能崩溃。

要避免此错误，请按照下列步骤执行操作：
 - 在主监视器中运行 LifeCam 软件。

3.0  视频捕获
---------------------

3.1  LifeCam VX-6000 -- 使用 USB 1.1 端口时不能支持所有分辨率

连接到 USB 1.1 端口时，LifeCam VX-6000 支持的最大视频分辨率为 800 x 600 像素，
支持的最大照片分辨率为 190 万像素。 

要解决此错误，请按照下列步骤执行操作：
- 如果可以，请连接到 USB 2.0 端口。
  - 或者 -
- 调低分辨率。

---------------------

3.2  LifeCam VX-6000 -- 130 万像素视频分辨率不支持平移、倾斜、缩放和脸部追踪

当前，在 130 万像素（最高）视频分辨率的情况下，平移、倾斜、缩放和脸部追踪功能不可用。
要使用平移、倾斜、缩放和脸部追踪，请选择较低的视频分辨率。

---------------------

3.3  LifeCam NX-6000 -- 预览捕获可能呈现黑色

在某些情况下，预览捕获可能无法显示。

要解决此错误，请按照下列步骤执行操作：
- 拔出网络摄像机，然后重新插入。
  - 或者 -
- 调低视频分辨率。

---------------------

3.4  当使用普通视频驱动程序时，捕获视频停留在前景中

在某些情况下，视频捕获将保持在前景中，即使您在 LifeCam 软件之上打开其他窗口。
如果为视频显示使用一般驱动程序，则可能发生这种情形。

要解决此错误，请按照下列步骤执行操作：
- 有关如何更新视频驱动程序的信息，请参阅硬件制造商文档。


---------------------

3.5  某些视频效果可能导致性能降低

当添加到高分辨率视频时，某些视频效果可能导致性能降低

要提高性能，请按照下列步骤执行操作：
- 将视频分辨率设置为较小值。
  - 或者 -
- 关闭视频效果。

4.0  即时消息程序
---------------------

4.1 网络摄像机上的“Windows Live Call”按钮无法正常运行

在某些情况下，安装或更新 LifeCam 软件期间，网络摄像机上的“Windows Live Call”按钮
可能无法正确初始化。 

要更正此问题，请按照下列步骤执行操作：

- 注销然后重新登录计算机。
  - 或者 - 
- 重新启动计算机。 

---------------------

4.2  LifeCam NX-6000 -- Microsoft LifeCam 软件对于某些消息客户端而言可能是必需的

要确保产品与某些消息客户端兼容，例如 Skype 和 Yahoo! Messenger，
可能需要 Microsoft LifeCam 软件。

---------------------

4.3 视频效果在所有即时消息程序中不工作

某些视频效果无法显示在即时消息程序（如 Skype）中。

---------------------

4.4  America Online (AOL) Instant Messenger 中的视频通话可能断开连接

如果您位于防火墙之后，与 AOL Instant Messenger 建立的视频通话开始之后不久即会断开连接。

有关调整防火墙或路由器设置的详细信息，请参阅 www.aim.com。

5.0  其他应用程序
-----------------------

5.1  某些本机 64 位程序不支持 LifeCam VX-6000 

LifeCam VX-6000 使用某些本机 64 位程序无法运行。 但是，大多数情况下，
在 64 位操作系统中使用这些程序的 32 位版本可以运行。

---------------------

5.2  LifeCam VX-6000 可能无法显示在 Microsoft Office Communicator 音频和视频调节向导中。 

在 Microsoft Office Communicator 中，通过音频和视频调节向导设置 LifeCam。
在某些情况下，LifeCam VX-6000 可能不会显示在选项列表中。

要解决此错误，请按照下列步骤执行操作：
-  在启动计算机之前插入 LifeCam 网络摄像机。

---------------------

5.3 当张贴到 Windows Live Spaces 时，我看到的是“已经登录”页面或 HTML 代码，而不是我的个人空间

如果您在同一台计算机上使用两个用户帐户，并将照片张贴到 Windows Live Spaces 帐户，
Windows Live Spaces 将出现“已经登录”错误。

要解决此错误，请按照下列步骤执行操作：
1.  关闭所有 Internet Explorer 浏览器窗口。 
2.  在 LifeCam 软件中，再次单击“张贴到我的 Windows Live Spaces 网络日志”按钮。 

>>要避免此错误，请按照下列步骤执行操作：

- 关闭用户会话之间的 Internet Explorer。

---------------------

5.4  如果某个用户更改了默认网络摄像机，则所有用户的默认网络摄像机均会被更改。

LifeCam 软件设置可能影响所有用户配置文件。例如，如果您在某个用户的配置文件中更改了
默认网络摄像机，则该计算机上所有用户配置文件的默认网络摄像机均会被更改。

---------------------

5.5  在 Windows XP 系统上 Microsoft LifeCam 软件中 LifeCam NX 6000 特定的某些功能可能会在网络摄像机“属性”窗口中出现截断

在 Windows XP 系统上 Microsoft LifeCam 软件中 LifeCam NX 6000 特定的某些功能可能会在网络摄像机“属性”窗口中出现截断，
该截断与 Windows UVC 驱动程序中的属性设置有关。但是，其相关的功能，如更改网络摄像机属性或将更改恢复到默认值，
仍可正常工作。

6.0  错误消息
---------------------

6.1  缺少“LifeCam 文件”文件夹时出现“硬盘空间不足”错误

如果“我的文档\LifeCam 文件”文件夹被删除，或者该文件夹的位置不可用
（例如，在网络共享上），您的视频剪辑将无法保存，并且可能会出现错误消息：
“可用硬盘空间不足，无法捕获照片、音频剪辑或视频剪辑。 请增加硬盘上的可用空间，
然后重试。” 

要解决此错误，请按照下列步骤执行操作：
- 重新创建“我的文档\LifeCam 文件”文件夹。
  - 或者 -
- 关闭然后重新打开 LifeCam 软件。系统将自动重新创建该文件夹。

---------------------

6.2  系统崩溃，Tiptsf.dll 出现严重错误

您的计算机可能收到严重错误（Tiptsf.dll 发生 STOP 错误）。 

要解决此错误，请按照下列步骤执行操作：
下载 Windows XP SP1 USB 1.1 修补程序。  
1.  转至 http://support.microsoft.com。
2.  搜索 KB827220。
3.  下载并安装修补程序。


---------------------

6.3  Windows XP SP1 中的 LifeCam VX-6000：Usbport.sys 出错，系统蓝屏

您的计算机可能收到严重错误（Usbport.sys 发生 STOP 错误）。 

要解决此错误，请按照下列步骤执行操作：
下载 Windows XP Service Pack 2 (SP2)。  
1.  转至 http://update.microsoft.com。
2.  下载并安装 Windows XP Service Pack 2 (SP2)。
